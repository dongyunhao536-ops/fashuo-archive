# 法律更新档案 — 2024年10月之后

## 用途
《2026法律(非法学)考试分析》于2025年10月出版,但其法律状态约截止于2024年8-9月(内容定稿滞后出版约一年)。
本目录收集 **2024年10月及之后** 颁布/施行的、与法硕(非法学)考试相关的法律法规和司法解释,作为教材的补充。
⚠️ **门槛于 2026-06-05 由"2025年10月之后"下调为"2024年10月之后"**;2024年10月~2025年9月这一年的法条/解释**尚未回填**,见文末待办。

## 答疑优先级补充规则
原 CLAUDE.md 答疑优先级:
> ① 心得文件 → ② 真题分析 → ③ 教材 → ④ 民法典/刑法条文 → ⑤ 司法解释

**新增**:遇到 2024年10月之后颁布的新法/新解释,若与教材冲突,**仍以教材为准**(因为考试以教材为准),但应**在答疑时主动提示新法变动**,告知用户"教材未更新,但实务/最新司法解释已变更为XXX"。

## 文件清单
- `01_刑法相关更新.md` — 贪污贿赂解释(二)、刑十二配套解释等
- `02_民法相关更新.md` — 民法典配套司法解释、监护人赔偿请求权等
- `03_宪法行政法相关更新.md` — 治安管理处罚法、监察法实施条例等
- `04_法理学相关更新.md` — 法治宣传教育法等
- `05_本年度立法动态总览.md` — 2025年10月-2026年6月人大常委会立法概览
- `_新法考点关联速查表.md` — 每条新法的考试关联度评估(★星级)

## 维护规则
1. 每发现一项新颁布的法律/司法解释,先评估是否与法硕考试科目(刑民法理宪法制史)相关,无关则跳过。
2. 每条记录必须包括:**颁布机关、文号、发布日期、施行日期、与教材的关系、考试关联度评估**。
3. 关联度评估星级:
   - ★★★★★ — 直接对应考试分析新增/修改的考点,必背
   - ★★★★ — 与教材现有考点有关,可能在简答/案例题中作为加分点
   - ★★★ — 间接相关,了解即可
   - ★★ — 同领域但非法硕考试范围,知道有此变动即可
   - ★ — 与法硕基本无关,仅备查
4. 教材已吸收的(2024年9月之前的内容),原则上不再收录,除非有特殊新解读。
