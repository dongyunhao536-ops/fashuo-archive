# 经验帖蓝本

## 用途
本目录收录从网络上甄选的 10 篇法硕(非法学)上岸经验帖,作为辅导云制定学习规划、调整方法的 **参考蓝本**。

## 与既有材料的关系

| 既有材料 | 性质 | 用途 |
|---------|------|------|
| 教材《考试分析》 | 知识源 | 答题依据 |
| 真题分析/做题心得 | 命题规律 | 判断"怎么考、怎么答" |
| 法律更新档案 | 时效补丁 | 教材后的新法 |
| **经验帖蓝本(本目录)** | **方法论源** | **判断"怎么学、怎么规划"** |

## 文件清单

- `00_经验帖原始摘要.md` — 10篇经验帖核心要点(分数/方法/时间/教训/资料)
- `00_适用于云的方法论提炼.md` — 结合云个人情况(在职/三战/6.5月/强记忆/弱理解/目标北大)的方法论沉淀
- `01_刑民学习方法精选.md` — 专门提炼刑法民法的学习路径
- `02_法综背诵方法精选.md` — 法理宪法法制史的背诵套路
- `03_答题模板与得分技巧.md` — 主观题/案例题/法条分析题的答题三段论

## 选帖原则
- ≥5篇北大经验(目标院校)
- ≥2篇在职经验(同处境)
- ≥1篇三战或失败教训(避坑)
- ≥1篇短期高强度冲刺(用户6.5月可参考)
- ≥1篇方法论专项(背诵/答题)

## 使用规则
1. 辅导规划阶段(每月初/每阶段切换时) → 必查 `00_适用于云的方法论提炼.md`
2. 提供具体方法时 → 引用具体经验帖(标注 E1-E10),增强说服力
3. 用户提出"我应该怎么背/怎么做案例题" → 优先查 `01-03` 专项文件
4. 经验帖与教材冲突时 → **教材为准**(应试)
5. 经验帖与做题心得冲突时 → **做题心得为准**(已有真题验证)

## 注意事项
- 经验帖仅作 **方法论参考**,非教材
- 不同年份分数线、报录比可能变化,以最新官方数据为准
- 经验帖中提到的辅导班、老师课程为他人选择,云可参考但不必照搬
