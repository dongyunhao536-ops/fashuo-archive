# 刑法民法学习方法精选

> 提炼来源:E1(北大415)、E2(北大413)、E4(北大397)、E10(清华394)、答题模板搜索

---

## 核心论点(10篇经验帖共识)

> **刑法民法的核心是"理解",不是"背诵"。**
> 这两科占 150 分,案例题/法条分析题占其中 60+ 分,**无法靠背诵拿到**。
> 如果你"背得很熟但案例不会做",就是没理解。

---

## 一、刑法学习路径(基于 E1/E2/E4)

### Step 1:听课/读分析建立框架(2-3周/编)
- **顺序**:总则 → 分则(财产犯罪 → 贪贿渎职 → 人身犯罪 → 公共安全 → 经济犯罪 → 其他)
- **方法**:听音频(1.5倍速)+ 同步看《考试分析》+ 标注重点
- **不必背**:这一步重在 **理解构成要件、罪与非罪的边界**

### Step 2:配套真题精做(章节级)
- 每学完1个分则章节,立刻做对应的真题(法硕近5年 + 司考刑分则)
- **错题反向定位**:错在哪个考点 → 回头读分析对应段落
- 建立 **错题本/刑法错题.md** 条目

### Step 3:罪名簇梳理(★ 关键步骤)
经验帖里讲"刑分则要系统学",但没人告诉你 **怎么系统**。
真正有效的方法是 **罪名簇**(已在 `02_刑法高频考点.md` 提到):

| 簇 | 包含罪名 | 区分要点 |
|---|---------|---------|
| 财产侵财簇 | 盗窃/抢劫/抢夺/诈骗/敲诈勒索/侵占 | 取得方式、暴力程度、占有转移时机 |
| 贪贿渎职簇 | 贪污/受贿/挪用公款/玩忽职守 | 主体身份、行为本质、对价关系 |
| 性侵簇 | 强奸/猥亵/聚众淫乱/组织卖淫 | 暴力胁迫、对象特定、人数 |
| 人身犯罪簇 | 故杀/故伤/故意致人重伤/过失致死 | 故意/过失、伤害程度、因果 |
| 共同犯罪簇 | 主从犯/胁从犯/教唆/帮助 | 作用大小、意思联络、独立性 |

### Step 4:案例题专项训练(10月开始)
- 用 `真题分析/_刑法做题心得.md` 已有规则
- 严格按 **三段论** 答(大前提-小前提-结论)
- 每周 2-3 道案例,每道至少 20 分钟思考+10 分钟改

## 二、民法学习路径(基于 E1/E2/E10)

### Step 1:听课/读分析(同刑法)
- **顺序**:民法总则 → 物权 → 合同 → 人格权 → 婚姻家庭 → 继承 → 侵权
- ⚠️ **云的特殊调整**:由于物权+合同是云的薄弱点,且也是命题富矿,**优先**

### Step 2:重点编精做(2026考点导向)
| 编 | 重要性 | 命题热点 |
|---|--------|---------|
| 物权 | ★★★★★ | 所有权(善意取得)/用益(土承包/建设用地)/担保(抵押/质押) |
| 合同 | ★★★★★ | 成立效力履行救济/买卖/赠与/租赁/借款/承揽 |
| 侵权 | ★★★★(2026新增监护人赔偿!) | 一般侵权/特殊侵权/产品责任/机动车交通事故 |
| 总则 | ★★★ | 民事法律行为/代理/时效 |
| 人格权 | ★★ | 一般人格权/具体人格权 |
| 婚姻家庭+继承 | ★★★ | 夫妻共同财产/法定继承 |

### Step 3:体系图(★ 关键步骤)
**民法最容易"看得懂,但忘得快"**。原因是知识点繁杂、规则交叉。
解决方法:**画体系图**(类似宪法八列表)。

**示例:物权变动体系图**
```
物权变动
├── 基于法律行为
│   ├── 不动产 → 登记生效(房屋买卖/抵押)/登记对抗(土承包/地役权)
│   └── 动产 → 交付生效(一般)/登记对抗(船舶/航空器/机动车)
├── 非基于法律行为
│   ├── 法院判决/继承/合法建造/拆除 → 直接生效,处分时需登记
│   └── 善意取得 → 占有+合理价格+登记/交付
```

### Step 4:案例题专项
- 民法案例题占比比刑法略低,但 **物权+合同+侵权三编必出**
- 用 `真题分析/_民法做题心得.md` 已有规则

## 三、刑民共通的"陷阱"

### 陷阱1:背诵不等于理解
- E9 失败者:"我背得很熟,但案例题一上手就懵"
- 解决:每章背完后,**自己出1道案例题考自己**

### 陷阱2:听课不等于学习
- E5 96天上岸者:"听课只是输入,真正学习是 **做题 → 错 → 翻书**"
- 解决:听课时长 ≤ 做题时长

### 陷阱3:用司考思维做法硕题
- E2 强调"理解达到司考高度",但 **答题以法硕教材为准**
- 法硕和司考观点冲突时,优先《考试分析》
- 司考真题只用于"练手感",答案以法硕版为准

### 陷阱4:错题不复盘
- 经验帖共识:**真题3-4遍中,后两遍重点是错题**
- 解决:`错题本/刑法错题.md` 和 `错题本/民法错题.md` 每周维护

## 四、推荐资料(经验帖共识)

### 必备
- **《考试分析》**(教材) — 10篇全提
- **历年真题**(法硕版,带答案解析)
- **基础配套练习**(章节练习题集)
- **法律法规汇编**(法条原文)

### 推荐
- **刑民司考真题**(精选,练理解)
- **背诵提纲**(网上有免费版,或买文运)
- **重要司法解释汇编**(2026更新见 `法律更新档案/`)

### 视频课(可选)
- 老师选择因人而异,核心看 **是否讲透构成要件**
- 法硕主流老师:刑法-车润海/陈永生、民法-岳业鹏/王斌
- 云在职建议:**只听精讲,不听通识** → 时间不够
