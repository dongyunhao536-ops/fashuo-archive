# 答题模板与得分技巧

> 提炼来源:E1 答题方法 + 法硕主观题模板搜索 + 真题分析做题心得
> 适用题型:简答题 / 法条分析题 / 案例分析题 / 论述题

---

## 总原则:阅卷采点给分(经验帖共识)

法硕阅卷是 **逐点给分**,不是看你写了多少字。
**关键词出现 = 得分;关键词缺失 = 不得分**。

所以:**结构清晰 + 关键词全 > 文字优美**

---

## 一、简答题模板(法理/宪/法制史/民总则)

### 标准结构(E1 总结)

```
【第一句】XX 是指 [严格按《考试分析》定义]
【分点列举】
  1. 第一点(关键词)— 展开 1-2 句
  2. 第二点(关键词)— 展开 1-2 句
  3. 第三点(关键词)— 展开 1-2 句
  [若必要,4-5点]
【收尾】[可选,总结性意义]
```

### 时间和字数
- **时间**:8-10 分钟/题
- **字数**:200-400 字(根据分值)
- **分点**:3-5 点最佳,**绝不超过 7 点**(超过反而显得没重点)

### 失分陷阱
- ❌ **概念不准** → 第一句的定义必须 **逐字背原文**
- ❌ **关键词缺失** → 分点的第一个词必须是 **教材原文术语**
- ❌ **展开过长** → 每点 1-2 句即可,展开过多反而稀释关键词密度

### 示例:简答"法治国家的特征"
```
【概念】法治国家是指法律具有最高权威,国家权力依法行使,公民权利得到充分保障的国家形态。
【分点】
  1. 良法的存在 — 即法律体系完备且合乎正义;
  2. 普遍守法 — 一切组织和个人都在法律范围内活动;
  3. 权力受制 — 国家权力的行使受到法律的有效制约;
  4. 司法独立 — 司法机关依法独立公正地行使审判权和检察权;
  5. 权利保障 — 公民的基本权利得到充分尊重和保障。
```

## 二、法条分析题模板(刑/民)

### 标准结构(基于觉晓法硕模板 + 真题分析)

```
【第一步:法条释义】
该条规定的是 XX 制度(/罪名/原则),具体含义是 [法条核心]。

【第二步:构成要件/适用条件】
本制度的成立需具备以下要件:
  1. 主体 — [具体描述]
  2. 主观方面 — [故意/过失/无要求]
  3. 客体/对象 — [具体描述]
  4. 客观方面 — [具体描述]

【第三步:法理意义/与他制度的区别】
本制度的设立旨在 [立法目的],
与 XX 制度(易混项)的区别在于 [关键区分]。
```

### 时间和字数
- **时间**:15-20 分钟/题
- **字数**:400-700 字

### 失分陷阱
- ❌ **照搬法条** → 必须 **解释**,不能复述
- ❌ **不分点** → 必须列点,阅卷老师才能采分
- ❌ **不区分易混** → 法条分析的高分点是 **辨析**,缺这一段顶多得6成分

## 三、案例分析题模板(刑/民)★ 重点

### 三段论(基于经验帖搜索 + 真题分析)

#### 刑法案例题

```
【问题罗列】(可选,题目复杂时先列出本题问几个问题)
问 1:XX 行为是否构成 XX 罪?
问 2:XX 与 YY 是否构成共犯?
问 3:XX 的犯罪形态是?

【针对每问的三段论】

▶ 大前提:XX 罪是指 [严格按教材定义],其构成要件包括:
  ① 主体 ② 主观方面 ③ 客体 ④ 客观方面 [若有特殊要件补充]

▶ 小前提:本案中,行为人 [姓名] 实施了 [具体行为]:
  ① 主体方面:符合(/不符合) — 因为 [事实]
  ② 主观方面:有 [故意/过失],体现为 [事实]
  ③ 客观方面:实施了 [构成要件行为],造成了 [危害结果]
  ④ 因果关系:[事实]导致[结果],存在因果关系
  
▶ 结论:综上,[姓名] 的 XX 行为构成(/不构成)XX 罪。

[若有犯罪形态、共犯、罪数等问题,继续分析]
```

**时间分配**(经验帖明示):
- 刑法案例:**35 分钟左右,不超过 40 分钟**
- 民法案例:**30-35 分钟**
- 一般案例:**25-30 分钟**

#### 民法案例题

```
【问题罗列】(同刑法)

【针对每问的请求权基础三段论】

▶ 大前提:根据《民法典》第 XX 条,XX 制度的规定是 [法条核心],
  其适用条件包括:① ② ③

▶ 小前提:本案中,[姓名] 与 [姓名] 之间形成 XX 法律关系,
  ① [条件一] — [事实分析]
  ② [条件二] — [事实分析]
  ③ [条件三] — [事实分析]

▶ 结论:综上,[姓名] 可以(/不能)主张 XX 权利,
  其请求权基础为《民法典》第 XX 条。
```

### 失分陷阱(★ 云必读)

#### 陷阱1:跳过大前提,直接写结论
❌ 错误:"小明构成盗窃罪,因为他偷了小红的手机"
✅ 正确:先讲盗窃罪定义和构成要件 → 再分析小明的行为 → 最后下结论

#### 陷阱2:小前提不联系大前提
❌ 错误:列了构成要件但不一一对应分析
✅ 正确:大前提说几个要件,小前提就分析几个要件,**一一对应**

#### 陷阱3:漏掉"罪数/形态/共犯"等附加问
❌ 错误:只判断了主罪,忘了"既遂未遂""主从犯""数罪并罚"
✅ 正确:对照刑法做题心得,**每道案例题都过一遍这些"附加问"清单**

#### 陷阱4:多人案件不分别分析
❌ 错误:把多个被告人混在一起讲
✅ 正确:**先分析每个人各自构成什么**,再讨论共犯关系

## 四、论述题模板(法理/宪法)

### 标准结构

```
【引言】(简短,1-2句)
[点明题目核心,给出立场]

【主体一:概念阐释】
[严格按教材定义,2-3句]

【主体二:理论分析】
[分点论述,每点结构:小论点+论据+分析]
  1. 点 1 — 论据 — 分析
  2. 点 2 — 论据 — 分析
  3. 点 3 — 论据 — 分析
  [3-5点]

【主体三:实践意义/现实关联】
[联系当前法治建设、习近平法治思想、具体立法实例]
  - 可引用 **《法治宣传教育法》**(2025-11-01施行,新论据!)
  - 可引用《民法典》具体条文
  - 可引用 **法释〔2026〕6号 贪污贿赂解释(二)** 体现平等保护

【结论】
[1-2句,呼应引言,深化主旨]
```

### 时间和字数
- **时间**:30-40 分钟/题
- **字数**:800-1500 字

### 加分点(经验帖共识)
- ✅ **引用新法/新解释** — 体现备考前沿(用 `法律更新档案/` 内容)
- ✅ **联系习近平法治思想** — 体现政治站位
- ✅ **法言法语** — 用 "应当/可以/不得" 而非 "应该/能/不能"
- ✅ **结构标号清晰** — 用 "首先/其次/再次/最后" 或 "(一)(二)(三)"

## 五、四段式作答(已在 CLAUDE.md 验证)

云已经接受过的"四段式"答题(来自之前的辅导工作流验证):

```
1. 心得文件引用(如有)— "根据真题分析,本题倾向于..."
2. 教材原文 — "《考试分析》第 XX 页规定..."
3. 法条/司法解释 — "《XX 法》第 XX 条规定..."
4. 案例应用/结论 — "结合本案,综上..."
```

此结构在辅导阶段使用,**正式考试时简化为三段论** (大前提-小前提-结论)即可。
