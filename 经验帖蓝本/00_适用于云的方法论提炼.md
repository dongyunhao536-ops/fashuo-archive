## 第二部分:从10篇经验帖中提炼的"云专属规则"

### Rule 1:刑民"理解为本",法综"背诵为本"(来自 E1/E2/E4/E10)

10篇里有8篇用了这个二分法。这是法硕方法论的"金标准"。

**对云的具体翻译**:
- 刑法、民法 → 必须先 **听课/读分析理解**,再背诵 → 否则就是云三次失败的老路
- 法理学、宪法、法制史 → **背诵为主**,理解辅助 → 这是云的强项,要稳住

**辅导触发规则**:
- 当云说"刑法/民法 XX 没听懂" → 必须停下来用案例 + 通俗解释,而不是让他直接背
- 当云说"法理学/法制史 XX 记不住" → 用记忆口诀 + 框架图,不必深入展开理解

---

### Rule 2:基础阶段必须在 9 月底完成(来自 E9 失败教训 + E1/E10 时间表)

E9 失败的核心原因是"9-10月才启动,基础来不及打牢"。
云三次失败的原因 **完全一致**。
所以这次的红线是:**9月30日基础阶段必须结业**。

**判断"基础结业"的标准**(经验帖里被多次提到):
- ✅ 《考试分析》已通读完,**重点章节(刑分、民物合、宪机构、立法法)精读**
- ✅ 每章节配套练习已做完一遍
- ✅ 近5年(2021-2025)真题刑民部分已精做并标错
- ✅ 法综高频考点已建立背诵提纲

**辅导触发规则**:
- 每月底自动检查上述4项的完成情况
- 8月底如果有2项未完成 → **预警**,必须调整投入
- 9月底如果有1项未完成 → **降级处理**(承认某些章节战略放弃)

### Rule 3:"短期高密度 + 在职碎片化"双轨节奏(来自 E5 + E7/E8)

E5(96天上岸)证明法硕"可以短期搞定",但前提是日均13h。
E7/E8 证明在职"每周25-30h也能上岸",但需要 ≥9 个月。
**云只有6.5个月+在职**,需要 **复合策略**:

**双轨节奏模板**(基于 E7/E8 调整,适用于云):

| 时段 | 任务类型 | 时长 |
|------|---------|------|
| 工作日早 6:30-7:30 | 法综背诵(强项,适合早上头脑清醒时记忆) | 1h |
| 工作日通勤(去+回) | 听刑民音频课/默背口诀 | 1-2h |
| 工作日晚 20:00-22:00 | 刑民精读(理解性内容,需大块时间) | 2h |
| 周六 9-12,14-18 | 真题精做 + 错题分析 + 法综背诵 | 8h |
| 周日 9-12,14-18 | 同上(周日下午留2h机动) | 6-8h |

**周合计**: 5×(1+1.5+2) + 8 + 7 = **约 37.5h** ← 超出云说的25-30h上限

**如果做不到37.5h的现实调整**:
- 工作日通勤只听不动手 = 1h算半数
- 早上1h保下来(对早起族友好)
- 晚上2h不能再少
- 周末必须保12-14h

**保守版周时数**: 5×(1+0.5+2) + 12 = **约 29.5h** ← 与云说的"上限"匹配

### Rule 4:背诵"四轮三阶段"法(来自 E1 + 艾宾浩斯方法)

E1 明确"背诵分三阶段,每阶段按 刑→民→法理→宪→法制史 循环"。
华图等专项资料里"40天10遍"法基于艾宾浩斯。

**对云的具体落地**:

| 轮次 | 时间窗口 | 范围 | 强度 |
|------|---------|------|------|
| 第1轮 | 7-8月 | 法理 + 法制史(强项先稳)+ 宪法骨架 | 慢,**理解记忆** |
| 第2轮 | 8-9月 | 刑民+全部法综 | 中,**框架记忆** |
| 第3轮 | 10-11月 | 全科,以分析为本 | **快速循环** |
| 第4轮 | 12月 | 高频考点 + 错题 | **冲刺记忆** |

**艾宾浩斯关键技巧**(来自背诵方法搜索):
- 背完1页立即复习该页 → 累计6页后回到第1页 → 累计12页一个循环
- 1天/2天/4天/7天/15天 各复习一遍 → 长期记忆

**辅导触发规则**:
- 云提到"XX背不下来" → 检查是否按上述节奏循环
- 月底检查背诵进度是否对得上轮次表

### Rule 5:"概念+分点"答题结构(来自 E1 + 答题模板搜索)

**简答题、法条分析题、论述题** 必须按统一结构:

```
【第一句】XX 是指 [概念,严格按《考试分析》]
【分点】
  1. 第一点(展开1-2句)
  2. 第二点(展开1-2句)
  3. 第三点(展开1-2句)
  [若有总结性意义,补一句]
```

**案例分析题** 三段论结构:
```
【大前提】XX 法律规定:[法条核心 + 构成要件]
【小前提】本案中,行为人 XX 行为符合/不符合 上述要件,具体表现为...
【结论】综上,行为人 XX 构成/不构成 XX 罪/责任,理由如上
```

**辅导触发规则**:
- 云每次答主观题都必须 **按此结构作答**,我严格按结构评分
- 不能"想到啥写啥",必须"先列结构,再填内容"

### Rule 6:每日复盘(来自 E5 的"睡前框架梳理")

E5(96天上岸者)明示:**每天睡前用框架梳理当天所学**。
这是所有高分者的隐性共性,云三次失败可能恰恰缺这一步。

**对云的具体规定**:
- 每天睡前 10 分钟,在 `进度记录/学习进度.md` 写3行:
  1. 今天学了什么(科目+章节)
  2. 哪一点最不懂/最薄弱(下次重点)
  3. 哪一点最有把握(下次只复习不重做)
- 每周日额外 30 分钟,把一周的"最不懂"汇总,看是否需要进 `薄弱知识点/当前弱项.md`

### Rule 7:择校"既然定了就狠"(来自 E9 失败 + E1 北大第一)

E9 警告:"择校不可激进"。
E1 反证:"目标定了就死磕"。
关键区别:**是否承认差距并制定补差方案**。

**对云的态度**:
- 不替云劝退北大(尊重云的目标)
- **但要求云接受3个现实**:
  1. 北大法学院 2024 复试线 369 → 云目标 375+ 留 6 分缓冲,合理
  2. 复试占比 50% → 11月前必须开始研究复试材料(英语+面试模拟)
  3. **9月底模拟分 ≥ 320** = 在轨道上,< 320 必须紧急调整投入
- B计划提前想好(中政大、人大、华东政法等同档可调剂院校)

### Rule 8:政治英语"不能两极化"(来自 E10 + 通识)

E10 明示:"政治9月启动,英语全程不断"。
经验帖共识:
- **英语** 从一开始就 1h/天(单词+1篇阅读),12月再总冲刺
- **政治** 6-8月听一遍音频铺底,9月正式启动,11月狂背肖四肖八

**对云的具体安排**(融入双轨节奏):
- **6-8月** 英语 1h/天(通勤听+晚饭后做阅读),政治 0.5h(听音频不背)
- **9-10月** 英语保持,政治升至 1.5h/天(精讲精练)
- **11-12月** 英语单词每天半小时维持,政治 2h/天(肖四肖八)

## 第四部分:与已有材料的接口

### 与 `真题分析/_刑法做题心得.md` 的关系
- 经验帖偏 **方法论**(怎么学),做题心得偏 **真题规律**(怎么答)
- 答疑时:做题心得优先(真题证据),经验帖作为方法论背书

### 与 `法律更新档案/` 的关系
- 经验帖讲"该学什么"(以教材为准),法律更新档案讲"教材之外要补什么"
- 答题以教材为准,经验帖与法律更新档案均不超越教材

### 与 `CLAUDE.md` 的辅导优先级
原优先级:① 心得文件 → ② 真题分析 → ③ 教材 → ④ 法条 → ⑤ 司法解释
**经验帖的位置**:**不进入答题优先级**(只作方法论参考),但进入 **规划优先级**:
- 规划层面:① 经验帖蓝本(本目录) → ② 真题分析(命题趋势) → ③ 教材(进度量)
- 答题层面:不变
