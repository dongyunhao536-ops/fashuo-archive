# 抢劫罪（致人重伤死亡）vs 故意伤害（致死）

> 📌 凡题干是"暴力 + 取财 + 出现伤亡结果，问定一罪还是数罪" → **必查本档案**。

---

## 一句话区分 test ★最重要★

> 看**暴力与取财的关系**：
> - 暴力是**当场取财的手段**（压制反抗→劫取）→ **抢劫罪一罪**，伤亡结果是"抢劫致人重伤、死亡"的**结果加重情节**，不另定伤害/杀人罪。
> - 暴力与取财**无手段关系**（先泄愤伤害，事后起意拿走财物）→ **故意伤害罪 + 盗窃罪**等数罪。
> - 盗窃/诈骗/抢夺后**为窝藏赃物、抗拒抓捕、毁灭罪证**当场使用暴力 → **转化型抢劫**（269 条），仍定抢劫一罪。

---

## 教材依据

- **考试分析_文本.txt 行 8053 起（P122）**：抢劫罪（2019 法/非法多，2021 法/非法单，2020 法案例）——以非法占有为目的，对财物的所有人、保管人**当场使用暴力、胁迫或其他方法**强行劫取财物；致人重伤、死亡是法定刑升格情形。
- **考试分析_文本.txt 行 7463 起（P115）**：故意伤害罪——致人死亡的，法定刑升格（结果加重犯）。
- **考试分析_文本.txt 行 3224 起（P57）**：结果加重犯的处理——只定基本罪名、适用加重法定刑，不数罪并罚。

## 关键对照表

| 维度 | 抢劫致人重伤死亡 | 故意伤害致死 + 取财 |
|---|---|---|
| 主观 | 非法占有目的 + 暴力手段故意 | 伤害故意；取财是**事后另起**犯意 |
| 暴力的功能 | 压制反抗以**当场**取财 | 与取财无手段关系 |
| 时空特征 | 暴力与取财"当场性" | 可分离（先伤害，后拿财） |
| 罪数 | 抢劫**一罪**（加重情节） | 伤害 + 盗窃（侵占）**数罪** |
| 死者财物 | 当场劫取=抢劫 | 杀伤后才起意拿走 → 盗窃（通说按占有状态判断） |

---

## 题型陷阱模式

1. **"为劫财把人打死"**：抢劫致人死亡一罪，选项"抢劫+故意杀人并罚"通常是坑（**预谋杀人灭口再取财**等特殊情形除外，按教材立场答）。
2. **"打完人见财起意"**：先伤害后另起盗心 → 数罪；不是抢劫（缺当场取财手段关系）。
3. **转化抢劫三前提**：先犯盗/骗/夺 + 当场暴力 + 三目的之一（窝赃/拒捕/毁证），三者缺一不转化。
4. **携带凶器抢夺** → 直接按抢劫定（267 条 2 款），不需实际使用。
