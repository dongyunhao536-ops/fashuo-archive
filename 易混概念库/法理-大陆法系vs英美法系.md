# 大陆法系 vs 英美法系（两大法系对比）

> 📌 高频考点 #9，别名配对题是送分也是丢分点。**必查本档案**。

## 一句话区分 test ★最重要★

> - **大陆法系**：以**罗马法**为基础、以**成文法典**为主要形式（1804《法国民法典》+1896《德国民法典》两大代表）——又称民法法系、罗马法系、法典法系、成文法系、罗马-德意志法系。
> - **英美法系**：以**英国普通法**为基础、以**判例法**为主要形式——又称普通法法系、判例法系、海洋法系、不成文法系。

## 教材依据

- **法理学_文本.txt 行 114**：大陆法系——以罗马法为基础，法、德民法典为代表；包括**法国法系**（1804，强调个人权利）与**德国法系**（1896，强调国家干预与社会利益）两支；日本、土耳其、美国路易斯安那州、加拿大魁北克省属大陆法系。
- **法理学_文本.txt 行 113**：英美法系——以英国中世纪普通法为基础；11 世纪诺曼征服后形成判例法，14 世纪后又有**衡平法**；英国法（不成文宪法、单一制、法院无司法审查权）与美国法（成文宪法、联邦制、法院有司法审查权）两大分支。

## 关键对照表

| 维度 | 大陆法系 | 英美法系 |
|---|---|---|
| 历史基础 | 罗马法 | 英国普通法（+衡平法） |
| 法的形式 | 成文法典 | 判例法为主（也有制定法） |
| 法的分类 | **公法/私法** | **普通法/衡平法** |
| 诉讼模式 | 纠问制（职权主义）色彩 | 对抗制（当事人主义）、陪审团 |
| 法官角色 | 适用法律（"自动售货机"喻） | **造法**（遵循先例 stare decisis） |
| 法律推理 | 演绎为主 | 归纳（区别技术）为主 |
| 法学教育 | 学术型 | 职业型（律师培养） |
| 特殊属地 | 日本、土耳其、路易斯安那、魁北克 | 英联邦国家；**苏格兰除外** |

## 题型陷阱模式

1. **别名配对**：海洋法系=英美；法典法系=大陆——选项故意错配。
2. **美国特例**：属英美法系但有**成文宪法+司法审查**；**路易斯安那州**反而属大陆法系。
3. **英国本土的苏格兰**不属普通法法系（行 113 明文"苏格兰除外"）。
4. **衡平法**是英美法系内部与普通法相对的概念——别拿来与大陆法系对比维度混搭。
5. 两大法系**融合趋势**：大陆法系重视判例、英美法系制定法增多——绝对化表述（"英美法系没有成文法"）必错。
6. 我国法律传统上接近大陆法系，但属**中国特色社会主义法系/法治体系**的独立表述——别直接归入大陆法系（按教材口径答）。

## 一句话口诀

> **"罗马成文是大陆（公私分），判例普通是英美（普衡分）；美国成文宪法能审查，魁北克路州反着站"**
