# 全国人大 vs 全国人大常委会（职权划分）

> 📌 国家机构是宪法⭐⭐⭐"重灾区"（高频考点文件原话：你的薄弱点）。凡题干"由谁修宪/立法/决定战争" → **必查本档案**。

## 一句话区分 test ★最重要★

> - **全国人大**：最高国家**权力**机关——管"最大的事"：**修改宪法**（2/3 多数）、制定修改**基本法律**（刑事、民事、国家机构等）、选举/决定最高国家机关领导人、决定**战争与和平**问题。
> - **常委会**：全国人大的**常设机关**——管"日常+补充"：**解释宪法和法律**、制定修改**基本法律以外的法律**、闭会期间**部分补充修改基本法律**（不得同基本原则相抵触）、批准条约、决定**全国总动员或局部动员**、决定**战争状态的宣布**（闭会期间遇武装侵犯等）。

## 教材依据

- **宪法学_文本.txt 国家机构章（第二节 全国人民代表大会及其常务委员会，约行 371 起）**：全国人大的性质地位（行 372 起）、职权（行 377 起）；常委会的组成任期与职权（行 384 起）。
- **宪法学_文本.txt 行 112-120**：宪法解释——我国属**立法机关解释体制**，宪法解释权属**全国人大常委会**。
- **宪法学_文本.txt 行 123-132**：宪法修改——修宪程序：常委会或 **1/5 以上**全国人大代表提议，全国人大以全体代表 **2/3 以上**多数通过。

## 关键对照表（职权配对——单选多选必背）

| 事项 | 全国人大 | 常委会 |
|---|---|---|
| 宪法 | **修改**（2/3） | **解释** + 监督实施（两者均监督） |
| 法律 | 基本法律的制定修改 | 其他法律 + 闭会期间补充修改基本法律（不抵触基本原则） |
| 法律解释 | — | ✔（立法解释） |
| 人事 | 选举主席/副主席、中央军委主席，**决定**总理（主席提名）、决定军委其他组成人员等 | 闭会期间决定部长等个别任免 |
| 战争 | 决定**战争与和平问题** | 闭会期间决定**宣布战争状态**、决定**全国/局部总动员** |
| 条约 | — | 决定批准/废除同外国缔结的条约和重要协定 |
| 监督 | 改变或撤销**常委会**不适当的决定 | 撤销国务院不适当的行政法规；撤销省级国家权力机关不适当的地方性法规 |
| 特别行政区 | 决定特别行政区的设立及其制度 | — |
| 紧急状态 | — | 决定**全国或个别省级**进入紧急状态（国务院决定省内**部分地区**） |

## 题型陷阱模式

1. **修宪 vs 释宪**：修改归大会、解释归常委会——最经典互换坑。
2. **基本法律**清单：刑法、民法、刑诉、民诉、选举法、地方组织法等——"修改刑法"闭会期间常委会只能**部分补充修改且不抵触基本原则**（刑法修正案由常委会通过即此机制）。
3. **战争与和平问题**（大会）vs **宣布战争状态/动员**（闭会期间常委会）——主体随"是否闭会"切换。
4. **紧急状态**三级口诀：全国/省级=常委会决定；省内部分地区=国务院。
5. 撤销链：大会撤常委会的；常委会撤**行政法规**和**地方性法规**——"常委会撤销部门规章"= 错（那是国务院的）。

## 一句话口诀

> **"大会修宪管战和、定基本法选大员；常委释宪释法律、动员条约日常管"**
