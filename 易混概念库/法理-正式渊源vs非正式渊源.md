# 法的正式渊源 vs 非正式渊源（法的渊源体系）

> 📌 法的渊源是法理高频考点 #2，多选重灾区。凡题干"判决书能不能引用 X" → **必查本档案**。

## 一句话区分 test ★最重要★

> - **正式渊源**：具有**明文规定的法律效力**、可直接作为裁判依据——宪法、法律、行政法规、地方性法规、自治条例单行条例、规章、国际条约等（**制定法序列**）。
> - **非正式渊源**：不具有明文效力但有**说服力**——**习惯、判例、政策、法理（学说）、公平正义观念**等。
> 速判：法官能否在判决主文里**直接援引**作裁判依据？能=正式；只能辅助说理=非正式。

## 关键对照表（当代中国法的正式渊源清单——多选必背）

| 渊源 | 制定主体 | 备注 |
|---|---|---|
| 宪法 | 全国人大（特别程序） | 根本法、最高效力 |
| 法律 | 全国人大（基本法律）/常委会（其他法律） | |
| 行政法规 | **国务院** | |
| 地方性法规 | 省级及**设区的市**人大及其常委会 | |
| 自治条例、单行条例 | 民族自治地方人大 | 可变通（不得违背法律基本原则） |
| 规章 | 部门规章（国务院部委）/地方政府规章 | |
| 经济特区法规、监察法规 | 授权制定 | 新增易考 |
| 国际条约与国际惯例 | 缔结/参加 | 声明保留的除外 |

**非正式渊源**：习惯（民法典 10 条：法无规定可适用习惯，**不得违背公序良俗**——习惯由此获得**有限的正式渊源地位**，注意表述）、判例/指导性案例（"应当**参照**"，非"依据"）、政策、法理学说。

## 题型陷阱模式

1. **指导性案例**：是"参照"不是"依据"——选项把指导性案例列入正式渊源 = 按教材口径为坑（其拘束力是事实上的）。
2. **习惯**：民法领域"法律没有规定的可以适用习惯" → 此时习惯有正式渊源属性；但作为一般法理分类仍在非正式序列——看题目问的是"民法的渊源"还是"法的正式渊源"。
3. **法的渊源 ≠ 法的形式**：渊源强调"来源/效力来源"，形式强调"表现形式"——教材将二者基本对应使用，但概念辨析题会考定义差异。
4. **村规民约、单位章程** → 不是法的渊源（非国家制定认可）。
5. 军事法规/监察法规是清单易漏项（行 210）。

## 一句话口诀

> **"能进判决主文的是正式；习惯判例学说撑说理；指导案例只'参照'"**
