# 盗窃罪 vs 诈骗罪（处分行为分水岭）

> 📌 侵财罪⭐⭐⭐区。凡题干"用了点骗术拿走财物" → 别见"骗"就选诈骗，**必查本档案**。

## 一句话区分 test ★最重要★

> 分水岭 = 被害人有没有**基于错误认识的处分行为（处分意识）**：
> - 被骗后**自愿交付/处分**财产 → **诈骗罪**（骗术是取财的直接原因）
> - 骗术只是**调虎离山/创造条件**，财物仍靠行为人**自行取走** → **盗窃罪**（间接正犯式窃取）
> 问一句："财物是被害人'给'的，还是行为人'拿'的？"

## 教材依据

- **考试分析_文本.txt 行 8202 起（P124）**：盗窃罪（2023 非法单，2024 非法单）——窃取他人占有的财物。
- **考试分析_文本.txt 行 8290 起（P125）**：诈骗罪（2020 法/非法单，2023 法/非法多，2024 非法单）——虚构事实、隐瞒真相，使被害人陷入错误认识而**处分**财产。

## 关键对照表

| 维度 | 盗窃罪 | 诈骗罪 |
|---|---|---|
| 取财方式 | 违背占有人意志**窃取** | 被害人基于错误认识**自愿处分** |
| 骗术的角色 | 辅助手段（支开人、借看一下） | 取财的**直接原因** |
| 处分意识 | 无 | **有**（知道自己在转移占有） |
| 既遂标准 | 失控+控制 | 取得财产（被害人处分完成） |

### 经典案型归类
| 案型 | 定性 | 理由 |
|---|---|---|
| 假装试穿衣服趁机跑掉 | 盗窃 | 店员只让"试穿"，无转移占有的处分意识 |
| "借手机打电话"趁机逃走 | 盗窃 | 短暂递交≠处分占有 |
| 调包验钞/以假换真 | 盗窃 | 被害人不知占有已转移 |
| 冒充收费员收停车费 | 诈骗 | 被害人基于错误认识自愿交付 |
| 偷换商家收款二维码 | 通说倾向盗窃（窃取应收款项）；有争议，法硕按教材/真题口径作答 | 商家无处分给行为人的意识 |
| 机器不能被骗：对 ATM 用拾得的卡取款 | 信用卡诈骗罪（法定特别规定） | 法条明文"冒用他人信用卡" |

## 题型陷阱模式

1. **见"骗"字就选诈骗** = 最大坑；先查处分行为。
2. **三角诈骗**（被骗人≠被害人，但被骗人有处分权限）→ 仍是诈骗；无处分权限的第三人被骗交出 → 盗窃（间接正犯）。
3. **诈骗数额**：以实际骗得为准；"找零诈骗"算差额。
4. 盗窃的特殊形态（多次/入户/携凶/扒窃）**不要求数额较大**——选项用"未达数额不构成"挖坑。

## 一句话口诀

> **"给的算骗，拿的算偷；骗只开路、偷手收尾的，还是偷"**
