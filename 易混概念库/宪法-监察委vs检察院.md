# 监察委员会 vs 人民检察院（+与纪委的关系）

> 📌 2018 修宪新机构，高频考点文件标注"新机构，高频"。凡题干"职务违法犯罪由谁管" → **必查本档案**。

## 一句话区分 test ★最重要★

> - **监察委**：国家**监察**机关——对**所有行使公权力的公职人员**（全覆盖）监督、**调查**职务违法和职务**犯罪**、处置；用"**留置**"不用刑诉强制措施；**不适用**刑事诉讼法（调查阶段）。
> - **检察院**：国家**法律监督**机关——**审查起诉、提起公诉**、批准逮捕、诉讼监督、**公益诉讼**；监察委调查终结的职务犯罪案件**移送检察院审查起诉**。
> 分工链：**监委调查 → 检察院审查起诉（可退回补充调查）→ 法院审判**。

## 教材依据

- **宪法学_文本.txt 国家机构章（监察委员会节）**：监察委的性质（国家监察机关）、产生（本级人大产生，对它负责受它监督）、领导体制（**上级监委领导下级监委**——双重从属中的垂直性强于一般机关）、监察官与留置措施。
- **宪法学_文本.txt 人民法院与人民检察院节**：检察院是国家的**法律监督机关**；上下级是**领导**关系（法院上下级是**监督**关系——又一经典对比）。

## 关键对照表

| 维度 | 监察委员会 | 人民检察院 |
|---|---|---|
| 宪法定位 | 国家**监察**机关 | 国家**法律监督**机关 |
| 管辖 | 所有行使公权力的**公职人员**职务违法+职务犯罪 | 审查起诉一切刑事案件；诉讼监督；公益诉讼 |
| 手段 | 监督、调查、处置；**留置**（不叫拘留/逮捕） | 批捕、公诉、抗诉 |
| 产生与负责 | 本级人大产生，对**本级人大及其常委会**和**上一级监委**负责 | 对本级人大及其常委会负责；检察系统上下级**领导**关系 |
| 与党的机构 | 与**纪委合署办公**（一套人马两块牌子，党纪+国法贯通） | — |
| 主任/检察长任期 | 监委主任连任限制同人大（每届同人大，连续任职不得超过两届） | 检察长同 |

### 三机关上下级关系对比（必背）
- 法院上下级：**监督**关系（审级独立）
- 检察院上下级：**领导**关系
- 监察委上下级：**领导**关系

## 题型陷阱模式

1. **"监察委对职务犯罪提起公诉"** → 错：监委只**调查**，公诉权专属检察院。
2. **留置 ≠ 逮捕/拘留**：留置是监察措施，不适用刑诉法——选项混用名称必错。
3. **监督全覆盖**：含国企管理人员、公办教科文卫单位管理人员、基层群众自治组织中**从事管理的人员**——"只监督党员干部"= 错（那是纪委口径）。
4. **法院上下级是监督不是领导**——三机关关系对比是多选钉子户。
5. 监察委**由本级人大产生**，但"对上一级监委负责"的垂直性——双重负责的表述要完整。

## 一句话口诀

> **"监委查（留置），检察诉（法律监督）；法院监督、检察领导、监委也领导"**
