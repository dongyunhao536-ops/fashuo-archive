# 无权代理 vs 表见代理

> 📌 代理章核心分叉。已有档案 [[民法-善意取得vs表见代理]]（跨制度对比），本档案管"代理内部"的分叉。**必查本档案**。

## 一句话区分 test ★最重要★

> 都是"没有代理权"，分水岭在**相对人有没有合理信赖的外观**：
> - **狭义无权代理**：无代理权 + 相对人**不构成合理信赖** → 效力**待定**（被代理人可追认；相对人可催告/善意者可撤销）。
> - **表见代理**：无代理权 + **客观上有权利外观**（授权书、印章、长期交易习惯）+ 相对人**善意无过失** → 代理行为**直接有效**，后果由被代理人承担。
> 表见代理 = 无权代理的"特殊有效形态"，保护交易安全。

## 教材依据

- **民法学_文本.txt 行 1732-1750（第四节 无权代理）**：无权代理三情形（未授权/越权/代理权终止后）；**效力未定**——被代理人追认则有效；不获追认时无权代理人对相对人承担责任；被代理人有**拒绝承认的形成权**。
- **民法学_文本.txt 行 1752-1759（▶表见代理）**：表见代理概念与构成——行为人无代理权，但**第三人客观上有理由相信其有代理权**，法律行为后果由**本人承担**。
- **民法学_文本.txt 行 1578**：限制行为能力人订立合同与无权代理（表见代理除外）同属效力待定序列。

## 关键对照表

| 维度 | 狭义无权代理 | 表见代理 |
|---|---|---|
| 效力 | **待定**（追认→有效；拒绝→对被代理人不生效力） | **直接有效**（被代理人不能拒绝） |
| 相对人主观 | 不论善恶（善意者才有撤销权） | **善意且无过失** |
| 权利外观 | 无 | **有**（介绍信、合同书、印章、授权终止未收回的委托书等） |
| 相对人的权利 | 催告（30 日）+ 善意相对人**撤销权**（追认前） | 直接要求被代理人履行 |
| 被代理人救济 | 拒绝追认即可 | 承担后向无权代理人**追偿** |

## 题型陷阱模式

1. **"公司收回授权但没收回盖章空白合同书"** → 离职员工用它签约 = 表见代理，公司得认账。
2. **相对人明知无代理权仍签约** → 连狭义无权代理的保护都打折：不适用表见代理，且无撤销权；行为人与相对人**按各自过错**承担责任。
3. **追认的方式**：明示或**默示**（开始履行视为追认）；"沉默"原则上视为**拒绝**追认（30 日催告期满未表示=拒绝）。
4. **撤销权行使时点**：只能在**被代理人追认之前**，且限**善意**相对人。
5. 表见代理由**相对人主张**——被代理人不能反过来主张表见代理把合同砸实给相对人。

## 一句话口诀

> **"无权待定等追认；外观+善意变表见，直接有效本人扛，扛完再找代理人算账"**
