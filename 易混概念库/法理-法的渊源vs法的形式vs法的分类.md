# 法的渊源 vs 法的效力 vs 法的分类（第五章三节辨析）

> 📌 第五章三个并列概念群（目录：渊源 P21 / 效力 P24 / 分类 P25），多选混搭出题。**必查本档案**。

## 一句话区分 test ★最重要★

> - **法的渊源**：法的**来源/出处**（谁制定、什么效力等级）——回答"这条规范从哪来"。
> - **法的效力**：法**对谁、何时、何地**有效（对人/空间/时间效力 + 溯及力）——回答"管不管得着"。
> - **法的分类**：按不同标准给法**贴标签**（成文/不成文、实体/程序、根本/普通、一般/特别、国内/国际）——回答"它是哪一类"。

## 关键对照表

### 法的效力（第二节）速查
| 维度 | 规则 |
|---|---|
| 对人效力 | 我国采**属地为主、属人与保护为补充**的折中原则 |
| 空间效力 | 域内 + 法定域外延伸（驻外使馆、船舶航空器） |
| 时间效力 | 生效（公布即生效/指定日期）、失效（明示废止/新法取代）、**溯及力：从旧兼从轻**（原则不溯及既往） |
| 效力等级 | 宪法 > 法律 > 行政法规 > 地方性法规/规章；**特别法优于一般法、新法优于旧法**（同一机关）；裁决机制：法律间不一致由全国人大常委会裁决等 |

### 法的分类（第三节）速查
| 标准 | 分类 |
|---|---|
| 创制与表现形式 | 成文法 / 不成文法（习惯法） |
| 内容 | 实体法 / 程序法 |
| 地位效力 | 根本法 / 普通法 |
| 适用范围 | 一般法 / 特别法 |
| 制定主体与适用主体 | 国内法 / 国际法 |
| （英美语境） | 普通法 / 衡平法；公法 / 私法（大陆语境） |

## 题型陷阱模式

1. **"不成文法 = 没有文字"** → 错：不成文法指**未经国家制定程序系统编纂**（判例法有文字仍归"不成文法"传统分类语境）。
2. **效力等级裁决**：部门规章与地方政府规章不一致 → **国务院**裁决；地方性法规与部门规章不一致 → 国务院提出意见，认为应适用部门规章的**提请全国人大常委会裁决**——主体链是多选大坑。
3. **"特别法优于一般法"限同一机关制定**——跨等级时按效力等级走。
4. **溯及力**："从旧兼从轻"是我国通行原则，民法领域有"有利溯及"例外（民法典时间效力规定）。
5. 渊源/效力/分类三组概念互相窜门：问"法的分类"却放渊源选项（如"行政法规"）——审清问什么。

## 一句话口诀

> **"渊源问出身，效力问管辖，分类贴标签；裁决链：常委会管法律、国务院管规章"**
