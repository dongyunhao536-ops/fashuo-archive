# 易混概念库

> **建库日期**：2026-06-03
> **目的**：法硕（非法学）"概念间混淆"的正面战场。每一对易混概念一个文件，
> 提供"一句话区分 test + 真题反例库 + 历次混淆记录"。

---

## 这个库的定位

法硕的失分大头不是"完全不会"，而是**概念间相互污染**：
- 想象竞合 vs 法条竞合 — 哪个特别法优先？
- 善意取得 vs 表见代理 — 都保护"善意第三人"？
- 不安抗辩权 vs 先履行抗辩权 — 谁先到期？

每出现一次"我以为是 X，结果答案是 Y" → 这里增一条记录。
长期下来，**这个库会逐渐变成云的"个人薄弱画像"**，比任何外部材料都精准。

---

## 文件命名规范

```
{科目}-{概念A}vs{概念B}.md
```

例如：
- `刑法-想象竞合vs法条竞合.md`
- `民法-善意取得vs表见代理.md`
- `民法-不动产登记vs动产交付.md`（多元辨析也可）

科目缩写：**刑法、民法、法理、宪法、法制史**。

## 自动维护规则（绑定到「法硕答疑」skill）

### 触发新建

1. **预检命中**：答疑预检 Step 3 发现"无命中" + 本题确实是辨析题 → 答完后建档
2. **用户暴露混淆**：用户在某题选错且错因是"两个概念分不清" → 立即建档
3. **真题精读发现**：精读真题时遇到反复挖坑的辨析点 → 主动建档

### 触发更新

1. **新一次出错**：同一概念再次混淆 → "历次混淆记录"+1
2. **新真题命中**：每发现一道相关真题 → 真题反例库表格+1
3. **教材更新**：教材改版后重新锚定行号

### 触发并档/拆档

1. 两个文件其实辨析同一组概念 → 并档
2. 文件涉及 ≥3 个概念且变复杂 → 拆档
