# 法律解释方法辨析（文义/体系/历史/目的/当然）

> 📌 高频考点 #10，材料题常给一段裁判说理问"用了什么解释方法"。**必查本档案**。

## 一句话区分 test ★最重要★

> 看解释者**拿什么当尺子**：
> - 拿**字面含义** → **文义解释**（语法解释、字面解释）——一切解释的**起点和边界**。
> - 拿**上下文/其他条文的关联位置** → **体系解释**。
> - 拿**立法史料、草案说明、当时背景** → **历史解释**（立法者原意）。
> - 拿**法律的目的、要保护的利益** → **目的解释**。
> - "举轻以明重、举重以明轻"的事理推论 → **当然解释**。

## 教材依据

- **法理学_文本.txt 行 431 起（一、法律解释的概念）**：法律解释的概念、特点与方法体系（第九章第三节，目录行 13：法律解释 P51）。
- 我国正式解释体制：立法解释（全国人大常委会）、司法解释（最高法、最高检）、行政解释——**正式解释**与学理解释（无拘束力）相对。

## 关键对照表

| 方法 | 尺子 | 裁判说理的典型句式 |
|---|---|---|
| 文义解释 | 词语的通常含义/专门含义 | "依照本条'XX'一词的通常理解……" |
| 体系解释 | 条文在整部法律中的位置、与相邻条文的关系 | "结合第 X 条、第 Y 条的规定……" |
| 历史解释 | 立法过程资料、修法对比 | "本条修订时删去了……可见立法者……" |
| 目的解释 | 立法目的、保护法益 | "本法旨在保护……故应理解为……" |
| 当然解释 | 事理强弱关系 | "禁止携带宠物入内，牲畜更不得入内" |
| 扩张/限缩解释 | 解释**结果**相对于字面的宽窄（与上述"方法"不同维度） | "此处'子女'应包括养子女（扩张）" |

### 体制辨析（多选高发）
| 解释类型 | 主体 | 效力 |
|---|---|---|
| 立法解释 | 全国人大常委会 | 与法律**同等**效力 |
| 司法解释 | 最高人民法院、最高人民检察院 | 普遍司法适用效力 |
| 行政解释 | 国务院及主管部门 | 行政执法适用 |
| 学理解释 | 学者等 | **无**法律拘束力（非正式解释） |

## 题型陷阱模式

1. **扩张/限缩 vs 文义/目的**：前者描述**结果**宽窄，后者是**方法**——"用目的解释得出限缩结论"两者并存，不互斥。
2. **文义优先**：各方法冲突时文义是边界——"解释不得超出文本可能含义"（罪刑法定语境尤甚）。
3. **当然解释**在刑法中入罪方向受罪刑法定限制——"举轻明重"入罪要谨慎，多用于出罪或民事。
4. **"两高"才有司法解释权**——地方法院、高级法院发布的文件不是司法解释。
5. 立法解释与法律同等效力——选项降格其效力是坑。

## 一句话口诀

> **"文义起点定边界，体系看邻居，历史问当年，目的问初心，当然比轻重"**
