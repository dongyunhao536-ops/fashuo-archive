# 法律事件 vs 法律行为（法律事实二分）

> 📌 法律关系产生变更消灭的"扳机"分类，单选常客。凡题干"X 引起了法律关系变化，X 属于什么" → **必查本档案**。

## 一句话区分 test ★最重要★

> 法律事实 = 能引起法律关系产生、变更、消灭的客观情况。按**是否以当事人意志为转移**二分：
> - **法律事件**：**不以**当事人意志为转移——绝对事件（自然灾害、人的出生死亡、时间经过）+ 相对事件（由"他人"行为引起，但对**本法律关系当事人**而言不可控，如第三人纵火烧毁标的物）。
> - **法律行为**（行为）：**以**当事人意志为转移——合法行为 + 违法行为（违法犯罪也是"行为"类法律事实！）。

## 关键对照表

| 维度 | 法律事件 | 法律行为（行为） |
|---|---|---|
| 意志关联 | 不以当事人意志为转移 | 以当事人意志为转移 |
| 细分 | 绝对事件（自然）/ 相对事件（他人行为引起） | 合法行为 / 违法行为 |
| 典型 | 死亡（继承开始）、出生（权利能力）、洪水（保险事故）、期限届满 | 签合同、立遗嘱、侵权、犯罪 |
| 一果多因 | 同一法律后果可由事件+行为复合引起（"事实构成"）：如取得房屋所有权=买卖合同（行为）+登记（行为） | |

## 题型陷阱模式

1. **"死亡"是事件**——即使是被害身亡：对继承法律关系而言，死亡本身是事件；**杀人行为**对刑事法律关系是行为。**同一案情对不同法律关系分别定性**，这是最大坑。
2. **违法行为也是法律行为类事实**——选项暗示"只有合法行为才算"= 错（注意：法理学的"行为"分类 ≠ 民法"民事法律行为"以合法为要件的旧表述）。
3. **战争、罢工**：对合同当事人而言属（相对）事件。
4. **时间的经过**（时效届满）= 事件。
5. **事实构成**：需要**数个**法律事实共同引起一个法律关系（结婚=合意+登记）——单选问"哪个选项需要事实构成"。

## 一句话口诀

> **"由不得你的是事件，由得了你的是行为；一个案子两条线，分别定性别打架"**
