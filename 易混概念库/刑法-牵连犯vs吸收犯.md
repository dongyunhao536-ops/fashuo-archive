# 牵连犯 vs 吸收犯

> 📌 凡题干是"数个行为分别触犯数罪名，问定几个罪" → 先分清是牵连还是吸收，**必查本档案**。

---

## 一句话区分 test ★最重要★

> - **牵连犯**：数行为分别独立成罪，行为间是**手段-目的 / 原因-结果**的牵连关系（如伪造公文去诈骗）→ **从一重处断**。
> - **吸收犯**：数行为属**同一犯罪过程的不同阶段/形态**，前行为是后行为的所经阶段或后行为是前行为的当然结果（如入户盗枪后私藏该枪）→ **重行为吸收轻行为**，只定吸收之罪。
> 核心差异：牵连犯的数行为**各自指向不同法益、本可独立评价**；吸收犯的数行为**针对同一对象、同一法益进程**。

## 教材依据

- **考试分析_文本.txt 行 3329**：处断的一罪主要有连续犯、牵连犯、吸收犯。
- **考试分析_文本.txt 行 3370 起（P59）**：牵连犯的概念——实施某个犯罪，作为该犯罪的**手段行为或结果行为**又触犯其他罪名。

## 关键对照表

| 维度 | 牵连犯 | 吸收犯 |
|---|---|---|
| 行为个数 | 数行为 | 数行为 |
| 行为关系 | 手段-目的 / 原因-结果 | 阶段-完成 / 当然经过 |
| 对象与法益 | 常不同（公文管理 vs 财产） | 同一对象、同一法益进程 |
| 处断 | 从一重（法律另有规定除外） | 吸收之罪一罪 |
| 例 | 伪造证件骗保险；假冒商标卖假货 | 制造毒品后持有该毒品；入室盗枪后藏匿该枪 |

---

## 题型陷阱模式

1. **"伪造 X 然后用 X 去骗"** → 牵连犯从一重，不并罚；选项给并罚多为坑（注意刑法分则**另有明文规定并罚**的除外，如 157 条武装掩护走私）。
2. **"制造后持有/藏匿同一违禁品"** → 吸收犯，只定制造。
3. **盗枪后另行倒卖**：倒卖是新的法益侵害 → 不再吸收，数罪并罚。
4. 法硕倾向：能用教材列举的典型例就套典型例，不展开学理争议。
