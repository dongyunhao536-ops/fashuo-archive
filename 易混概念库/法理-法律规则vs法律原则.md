# 法律规则 vs 法律原则

> 📌 法的要素高频考点（高频考点文件 #3/#4）。凡题干"某条文是规则还是原则/冲突时怎么适用" → **必查本档案**。

## 一句话区分 test ★最重要★

> - **法律规则**：有**确定的行为模式 + 法律后果**（假定条件、行为模式、法律后果三要素）——适用是"**全有全无**"：要么适用，要么不适用。
> - **法律原则**：不预设具体行为模式与后果，只提供**方向性的价值准则**（如诚实信用、公序良俗）——适用是"**权衡**"：可与其他原则同案并存、分量相较。

## 教材依据

- **法理学_文本.txt 行 255 起**：法律的构成要素——法律概念、法律规则、法律原则（本节依次展开三要素及其逻辑结构）。
- 规则的逻辑结构：假定条件 + 行为模式（可为/应为/勿为）+ 法律后果（合法后果/违法后果）。
- 原则适用条件（通说，教材立场）：**穷尽规则方得适用原则**；**法律目的实现**所必需；**更强理由**才能舍规则用原则。

## 关键对照表

| 维度 | 法律规则 | 法律原则 |
|---|---|---|
| 内容 | 明确具体（三要素齐备） | 概括抽象（价值导向） |
| 适用方式 | 全有或全无 | 权衡分量 |
| 冲突解决 | 一条有效则另一条被排除（特别法优于一般法等） | 同案可并存，比重不同 |
| 覆盖面 | 针对**特定**行为类型 | 跨领域贯通（如平等原则） |
| 稳定性 | 易随社会变化修改 | 相对恒定 |
| 功能 | 提供确定指引（可预测性） | 弥补规则漏洞、统摄解释方向 |

### 分类速记（多选高发）
- 规则按**行为模式**分：授权性规则 / 义务性规则（命令性+禁止性）。
- 规则按**确定性程度**分：确定性规则 / 委任性规则（授权某机关规定）/ 准用性规则（"参照 X 执行"）。
- 原则按来源分：公理性原则（诚信、平等）/ 政策性原则（计划生育→优化生育等随政策变动）。

## 题型陷阱模式

1. **"参照前条规定执行"** = **准用性**规则（不是委任性——委任是"由 XX 机关另行规定"）。
2. **规则与原则冲突**：不能直接"原则优先"——须满足"穷尽规则 + 更强理由"才舍规则适用原则（泸州遗赠案是经典材料）。
3. **"禁止权利滥用"是规则还是原则** → 原则（无确定行为模式与后果）。
4. 材料分析题套路：先指出条文属规则/原则 → 写明逻辑结构或价值功能 → 再谈适用方式差异。

## 一句话口诀

> **"规则三件套、全有全无；原则讲分量、穷尽规则才出手"**
