# 疏忽大意的过失 vs 意外事件

> 📌 "没预见到结果"的两种定性，一罪一无罪，分界极细。凡题干"没想到会发生" → **必查本档案**。

## 一句话区分 test ★最重要★

> 都是**事先没有预见**，唯一分水岭：**应当预见且能够预见吗？**
> - 应当预见 + 能够预见，因疏忽没预见 → **疏忽大意的过失**（构成犯罪，法律有规定的）
> - 根本**不可能预见**（不能预见的原因引起）→ **意外事件**（无罪过事件，不是犯罪）
> 判断"能否预见"：以行为人本人的知识水平、经验、行为时的客观环境综合判断。

## 教材依据

- **考试分析_文本.txt 行 1516-1521**：无罪过事件 = 意外事件、不可抗力（"不是犯罪"）；意外事件三特征。
- **考试分析_文本.txt 行 1685-1692**：疏忽大意的过失——应当预见 + 因疏忽没有预见。
- **考试分析_文本.txt 行 1737-1751**：疏忽大意的过失与意外事件的异同——区别就在**对危害结果是否应当预见**；教材车夫马惊案例（行 1751）：对火灾无预见可能 → 意外事件。

## 关键对照表

| 维度 | 疏忽大意的过失 | 意外事件 |
|---|---|---|
| 事先预见 | 没预见 | 没预见 |
| 预见可能性 | **应当且能够**预见 | **不可能**预见 |
| 法律评价 | 有罪过 → 法律有规定的构成过失犯罪 | 无罪过 → **不是犯罪** |
| 同类项 | 与不可抗力相对照：不可抗力是"预见了但**无法抗拒**" | 意外事件是"**无法预见**" |

## 题型陷阱模式

1. **职业身份抬高预见义务**：司机/医生/电工对本行业风险"应当预见"——同一行为普通人可能是意外事件，专业人员就是疏忽过失。
2. **"被害人特异体质"**（轻推倒地死亡）：一般人无法预见特异体质 → 多按**意外事件**处理；但若已知对方有病仍为之 → 过失甚至间接故意。
3. **意外事件 vs 不可抗力**：选项互换"不能预见/不能抗拒"的表述——意外=不能**预见**；不可抗力=已预见但不能**抗拒**。两者都无罪。
4. 别把"结果严重"当入罪理由——再严重，不可预见就是意外事件。

## 一句话口诀

> **"该见没见是过失，见不了的是意外；见了挡不住，叫不可抗力"**
