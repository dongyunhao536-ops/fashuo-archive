# 高空抛物罪 vs 过失致人死亡罪 vs 过失以危险方法危害公共安全罪

> 📌 **刑修十一新增罪名（高空抛物罪）+ 过失犯罪的三角辨析**。
> 凡题干涉及"扔/抛/抛掷+从高处+砸伤"的案例 → **第一时间套此档案**。

---

## 一句话区分 test ★最重要★

> **高空抛物罪 = 从建筑物或者其他高空抛掷物品 + 情节严重**（刑法第 291 条之二，刑修十一新增）
> **过失致人死亡罪 = 过失致一人或数人死亡（特定对象）**
> **过失以危险方法危害公共安全罪 = 过失行为对不特定多数人有危险 + 实害结果**

**判断流程**：
1. **是从高空抛物吗？** → 是 → 满足高空抛物罪的客观要件
2. **是否情节严重？**（致人轻伤以上 / 多次 / 经处罚再犯 / 公共场所等）→ 是 → 构成高空抛物罪
3. **同时造成死亡结果吗？** → 是 → 与过失致人死亡罪**想象竞合**
4. **是否对不特定多数人有危险？** → 是 + 过失致死 → 过失以危险方法危害公共安全罪
5. **想象竞合 → 按重罪处罚**

---

## 共通点

1. 都可能涉及"扔东西/抛物 + 致人伤亡"
2. 都不要求行为人主观上有杀意（区别于故意杀人罪）
3. 都属侵害人身/公共安全类犯罪

## 教材依据

### 高空抛物罪（**法律更新档案**）
**《刑法》第 291 条之二**（刑修十一新增）：
> "从建筑物或者其他高空抛掷物品，情节严重的，处一年以下有期徒刑、拘役或者管制，并处或者单处罚金。
> 有前款行为，同时构成其他犯罪的，依照处罚较重的规定定罪处罚。"

⭐ **第二款是关键**：与其他罪想象竞合时**择一重**——这是命题常考点。

详见 `法律更新档案/01_刑法相关更新.md`。

### 过失致人死亡罪
**《考试分析》第 1084 行**："过失致人死亡罪，必须发生**死亡结果**才能构成该罪。"
**《考试分析》第 1678 行**：法定最高刑 7 年有期徒刑。

### 过失以危险方法危害公共安全罪
**《刑法》第 115 条第二款**：过失以危险方法危害公共安全，致人重伤、死亡或者使公私财产遭受重大损失的，处三年以上七年以下有期徒刑；情节较轻的，处三年以下有期徒刑或者拘役。
- 区别于第 114 条（具体危险犯，故意） 
- 第 115 条第 2 款是结果犯（过失 + 实害结果）
- **核心要件：对不特定多数人有危险**

## 三罪并列对照表

| 维度 | 高空抛物罪 | 过失致人死亡罪 | 过失以危险方法危害公共安全罪 |
|---|---|---|---|
| 主观面 | 故意（明知是高空抛物）| 过失（应当预见而未预见 / 已预见轻信避免）| 过失（轻信能避免对不特定多数人的危险）|
| 客观面 | 从建筑物/高空抛掷 + 情节严重 | 行为 + 死亡结果 | 行为 + 对不特定多数人危险 + 实害（重伤/死亡） |
| 既遂标准 | 情节严重即既遂 | 死亡结果发生 | 重伤/死亡/重大损失 |
| 法定刑 | **1 年以下**有期/拘役/管制+罚金 | 3-7 年；情节较轻 3 年以下 | 3-7 年；情节较轻 3 年以下 |
| 想象竞合 | **必然存在**（与他罪择一重） | — | — |

## 案例 57 的完整分析模板（可直接套用）

针对"扔物砸死路人"类案例，**法硕标准答题模板**：

**Step 1：高空抛物罪判断**
- 行为：从高楼/阳台抛掷物品 ✓
- 情节严重判断（致死=严重）✓
- → **构成高空抛物罪**

**Step 2：主观心态判断（区分故意/过失）**
- 是否查看楼下？查看过 → 无故意 / 无放任 → 排除故意杀人罪
- 是否每次都查看？只查看第一次而后续未查→ 对后续次数有**过失**
- → 过失致人死亡罪

**Step 3：是否构成过失以危险方法危害公共安全罪？**
- 案发地点是公共场所？农村清晨屋檐下 → **不属于不特定多数人**
- → **排除该罪**

**Step 4：罪数处理**
- 高空抛物罪 + 过失致人死亡罪 → **想象竞合**
- 按《刑法》第 291 条之二第 2 款"**处罚较重的规定**" → 过失致人死亡罪（最高 7 年 > 高空抛物罪最高 1 年）
