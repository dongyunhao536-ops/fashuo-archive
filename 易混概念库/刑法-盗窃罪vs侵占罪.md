# 盗窃罪 vs 侵占罪

> 📌 凡题干涉及"拿走别人财物，问盗窃还是侵占" → 先问**财物在谁的占有之下**，**必查本档案**。

---

## 一句话区分 test ★最重要★

> **盗窃 = 打破他人占有**（财物在他人占有下，秘密窃取）；
> **侵占 = 不打破占有**（财物已在自己合法占有/持有之下——代为保管物、遗忘物、埋藏物——变占有为所有，**拒不退还/拒不交出**）。
> 判断顺序：①财物当时归谁占有？②行为是"取得"还是"侵吞"？③侵占还要求"拒不退还"要件。

---

## 教材依据

- **考试分析_文本.txt 行 8202 起（P124）**：盗窃罪（2023 非法单，2024 非法单）——以非法占有为目的，窃取他人占有的公私财物。
- **考试分析_文本.txt 行 8335 起（P126）**：侵占罪——将代为保管的他人财物非法占为己有，数额较大**拒不退还**；他人的遗忘物、埋藏物同。
- **考试分析_文本.txt 行 8340（P126）**：职务侵占罪——利用职务上的便利侵占本单位财物（与普通侵占的分界：职务便利）。

## 关键对照表

| 维度 | 盗窃罪 | 侵占罪 |
|---|---|---|
| 占有归属（行为时） | **他人占有** | **自己占有**（保管/遗忘物/埋藏物） |
| 行为方式 | 窃取（打破占有建立新占有） | 侵吞（变合法持有为非法所有） |
| 特别要件 | 数额较大或多次/入户/携凶/扒窃 | 数额较大 + **拒不退还/拒不交出** |
| 告诉形式 | 公诉 | **告诉才处理**（亲告罪） |
| 既遂标准 | 失控+控制（通说取得说） | 拒不退还时 |

### 占有归属的细规则（高频陷阱区）
- 遗忘在**特定场所**（出租车、餐厅座位）的财物 → 转归**场所管理者占有** → 第三人拿走 = **盗窃**；管理者侵吞 = 侵占。
- **封缄物**：受托人对包装整体合法占有（侵吞整体=侵占），对**内容物**无占有（取出内容物=盗窃）。
- 死者身上财物：当场劫杀后取财以外，第三人路过拿走 → 通说按占有延续保护，定盗窃。

---

## 题型陷阱模式

1. **"乘客遗忘在出租车上的手机被下一位乘客拿走"** → 盗窃（司机占有），不是侵占遗忘物。
2. **"保管人把寄存的相机卖掉"** → 侵占；**"保管人撬开寄存的密码箱取现金"** → 盗窃（封缄物规则）。
3. **"拒不退还"**：侵占独有要件——题干出现"归还了"则侵占不成立。
4. 亲告罪：选项考"侵占罪由被害人自诉" = 正确表述。

## 一句话口诀

> **"先问占有在谁手：在他手是盗，在我手是侵；侵占还要赖着不还"**
