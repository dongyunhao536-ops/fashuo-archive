# 按份共有 vs 共同共有

> 📌 共有形态二分，"处分要多少人同意"是常考算术题。**必查本档案**。

## 一句话区分 test ★最重要★

> - **按份共有**：对共有物**按确定份额**享有权利、承担义务——份额可转让，转让时其他共有人**同等条件优先购买**。
> - **共同共有**：基于**共同关系**（婚姻、家庭、共同继承未分割）**不分份额**地共同享有——共同关系存续期间原则上**不得请求分割**。
> - **推定规则**：对共有形态没有约定或约定不明，**除具有家庭关系等外，视为按份共有**。

## 教材依据

- **民法学_文本.txt 行 2364-2378（二、按份共有）**：按份共有的概念、特征与权利——①依份额享有共有权 ②（约定不得分割但有**重大理由**可请求分割；无约定可**随时**请求分割）③份额转让时其他共有人**优先购买权**（同等条件，转让条件应及时通知）。
- 共同共有与管理处分规则（本节后续）：处分共有物及对共有物作**重大修缮、变更性质用途**——按份共有需占份额 **2/3 以上**的共有人同意；共同共有需**全体**共有人同意（另有约定除外）。

## 关键对照表

| 维度 | 按份共有 | 共同共有 |
|---|---|---|
| 份额 | 确定份额 | 不分份额 |
| 基础 | 不要求特定关系 | **共同关系**（夫妻、家庭、共同继承） |
| 处分/重大修缮 | 份额 **2/3 以上**同意 | **全体一致**同意 |
| 分割时点 | 无约定可**随时**请求；有约定但有重大理由也可 | 共同关系**消灭**或有重大理由才可分割 |
| 份额转让 | 自由转让 + 其他共有人**优先购买权** | 无份额可转 |
| 对外责任 | 对外连带（债权人可向任一共有人主张），对内按份额追偿 | 对外连带，对内平均/按约定 |
| 推定 | **默认形态**（非家庭关系时） | 须有共同关系支撑 |

## 题型陷阱模式

1. **"三人合伙买房各出 1/3，其中一人擅自卖整套房"** → 处分需 2/3 份额同意——1/3 不够；但**善意第三人**可能善意取得，注意两层分析。
2. **优先购买权**只在**份额转让**时存在；共有物**整体**出卖给第三人时按处分规则，承租人/共有人优先权竞合时**共有人优先**。
3. **夫妻共有**：一方擅自处分共有房屋，另一方原则可追回，但善意取得例外。
4. 推定方向别记反：**视为按份**（除家庭关系等）——选"视为共同共有"是坑。
5. 分割请求权：按份"随时可请求"（无约定时）vs 共同"关系存续中原则不可"。

## 一句话口诀

> **"按份三分之二动产权、份额能卖邻先买；共同全员点头、散伙才分；没约定按按份"**
