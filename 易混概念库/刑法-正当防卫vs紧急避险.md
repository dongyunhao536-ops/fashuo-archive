# 正当防卫 vs 紧急避险

> 📌 总论每年必考区。凡题干"面对危险/侵害做出反击或避让" → **必查本档案**。

## 一句话区分 test ★最重要★

> **防卫 = 正对不正**（反击不法侵害人本人，无限度内不要求别无选择）；
> **避险 = 正对正**（损害无辜第三人的合法权益保自己，**必须是不得已**=别无选择，且所损利益**必须小于**所保利益）。
> 速判三连：①危险来源是"人"的不法侵害还是任意危险？②损害对象是侵害人还是无辜第三者？③有没有"不得已"？

## 关键对照表

| 维度 | 正当防卫 | 紧急避险 |
|---|---|---|
| 危险来源 | 人的**不法侵害** | 不限（自然灾害/动物/人的侵害均可） |
| 损害对象 | **不法侵害人本人** | **无辜第三人**的合法权益 |
| 补充性 | 不要求（可以防卫也可以跑） | **必须不得已**（有其他办法就不成立） |
| 限度 | 不能明显超过必要限度造成重大损害；**特殊防卫**（行凶杀人抢劫强奸绑架等严重危及人身安全的暴力犯罪）**无限度** | 所损利益 < 所保利益（生命>健康>自由>财产；**不能牺牲他人生命保自己**） |
| 主体限制 | 任何人 | **职务上业务上负有特定责任的人**不得为避免本人危险而避险 |
| 过当责任 | 防卫过当：应当**减轻或免除** | 避险过当：应当**减轻或免除** |

## 题型陷阱模式

1. **"狗咬人"**：主人唆使狗咬 = 不法侵害的工具 → 打死狗是**正当防卫**；野狗自发咬 → 打死狗（毁第三人财产则是避险，打野狗本身无主物谈不上避险）。
2. **"开车撞向无辜路人躲避追杀"** → 紧急避险审查"不得已"+限度；牺牲他人生命 = 过当。
3. **消防员逃离火场不救火** → 职务上有特定责任，不得避险。
4. **防卫不要求"不得已"**——选项说"能逃跑就不能防卫"= 错。
5. 特殊防卫的罪名清单（行凶、杀人、抢劫、强奸、绑架）要背全，**盗窃不在内**。

## 一句话口诀

> **"防卫打坏人不讲退路；避险伤好人必须无路，还得小损换大保"**
