# 物权请求权 vs 债权请求权

> 📌 "请求返还/赔偿走哪条路"的体系题，与诉讼时效适用直接挂钩。**必查本档案**。

## 一句话区分 test ★最重要★

> - **物权请求权**：基于**物权的支配力**派生——返还原物、排除妨害、消除危险（教材口径将此三种归物权请求权）；以**物权存在**为前提，**随物权移转**，多数**不适用诉讼时效**。
> - **债权请求权**：基于**债的关系**（合同/侵权/不当得利/无因管理）——给付、赔偿损失等；**适用诉讼时效**（3 年）。
> 速判：主张的内容是"**把物的圆满状态恢复给我**"还是"**让你给我点什么**（钱/行为）"？

## 教材依据

- **民法学_文本.txt 行 2127**：返还原物、排除妨害、消除危险三种请求权属**物权请求权**。
- **民法学_文本.txt 行 2104（第五节 物权的保护）**：物权保护的方法体系。
- **民法学_文本.txt 行 1958**：物权的保护偏重"物上请求权"方法（与债权保护方法的对照）。
- **民法学_文本.txt 行 1838**：不适用诉讼时效的请求权——停止侵害、排除妨碍、消除危险；**不动产物权和登记的动产物权**的返还财产请求权。

## 关键对照表

| 维度 | 物权请求权 | 债权请求权 |
|---|---|---|
| 基础 | 物权（对世权） | 债（对人权） |
| 内容 | 返还原物 / 排除妨害 / 消除危险 | 履行、赔偿损失、支付违约金、返还不当得利等 |
| 前提 | 物权仍存在（物未灭失） | 债已发生 |
| 诉讼时效 | 排除妨害/消除危险**不适用**；不动产及登记动产的返还**不适用**；未登记动产返还**适用** | **适用**（3 年） |
| 过错要求 | **不要求**相对人有过错 | 侵权赔偿一般要求过错 |
| 物灭失后 | 转化为债权请求权（赔偿损失） | — |

## 题型陷阱模式

1. **"房子被占 20 年才起诉返还"** → 不动产返还请求权**不罹于时效**，能要回——选"超过最长 20 年丧失"是坑。
2. **物被毁损灭失**：物权请求权没了（无物可返）→ 只能主张**损害赔偿**（债权请求权，有时效）。
3. **赔偿损失是不是物权请求权？** 教材口径：返还/排除/消除三种是物权请求权；修理重作更换、赔偿损失属**债权性**保护方法——多选题按教材三分法。
4. 物权请求权**不要求过错**——"无过错的妨害人也要排除妨害"，选项以"他没过错"否定请求 = 错。
5. 占有人的**占有保护请求权**（返还占有物，1 年除斥期间）≠ 所有权人的返还原物请求权——主体与期间都不同。

## 一句话口诀

> **"物权护物不问错、多数不怕拖；债权要给付、三年要记牢；物毁权转债"**
