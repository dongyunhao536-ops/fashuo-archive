# 奴隶制五刑（旧五刑）vs 封建制五刑（新五刑）

> 📌 法制史最基础的贯穿性考点，五个名目+确立时间点单选直考。**必查本档案**。

## 一句话区分 test ★最重要★

> - **奴隶制五刑（旧五刑）**：**墨、劓、剕（刖）、宫、大辟**——以**肉刑**为核心（毁伤身体），夏商西周通行。
> - **封建制五刑（新五刑）**：**笞、杖、徒、流、死**——以**劳役刑+身体刑**替代肉刑，**《开皇律》（隋）正式确立**，唐沿用并完善。
> 转折枢纽：**汉文帝、景帝废肉刑**（文景刑制改革）开启旧→新过渡；魏晋南北朝逐步定型；隋《开皇律》最终确立。

## 教材依据

- **法制史_文本.txt 行 187-189（▶《开皇律》）**：《开皇律》以《北齐律》为基础，**确立封建制五刑**（笞杖徒流死）、十二篇体例。
- **法制史_文本.txt 行 230（▶五刑制度）**：隋律确立五刑制度后唐律的继承与具体刑等。
- 文景刑制改革段（汉律部分）：废肉刑——黥（墨）改髡钳城旦舂、劓改笞三百、斩左趾改笞五百（景帝又减笞数、定箠令）。

## 关键对照表

| | 奴隶制五刑 | 封建制五刑 |
|---|---|---|
| 名目 | 墨（黥面）、劓（割鼻）、剕/刖（断足）、宫（去势）、大辟（死刑） | 笞（10-50）、杖（60-100）、徒（1-3 年，分五等）、流（二千/二千五/三千里）、死（绞、斩） |
| 性质 | 肉刑为主（不可恢复的毁伤） | 劳役刑+身体刑+生命刑的等级序列 |
| 时代 | 夏商西周春秋 | 隋确立 → 唐完善 → 沿用至清 |
| 确立标志 | — | **《开皇律》** |

### 过渡链（排序题必背）
旧五刑 → **文景废肉刑**（汉）→ 魏晋北朝调整（北齐"重罪十条"同期）→ **《开皇律》定型新五刑** → 唐律承之 → （宋有凌迟入律《庆元条法事类》；明清五刑外有充军、发遣、刺字等闰刑）。

## 题型陷阱模式

1. **新五刑确立于《开皇律》**——选"唐律""北齐律"都是坑（北齐律贡献的是"重罪十条"与十二篇化进程）。
2. **文景改革废的是肉刑**（墨劓剕），**宫刑后又有反复**、大辟保留——"文景废除全部旧五刑"= 错。
3. **死刑二等**：唐制**绞、斩**（绞轻斩重，留全尸为轻）；**凌迟**是五刑之外的法外刑入律（宋）——朝代归属抠细节。
4. **流刑三等**里程数（二千/二千五百/三千里）+ 唐加役流（贞观，流三千里居作三年，作为死刑减等）。
5. 明清的**充军、发遣、刺字**是五刑之外的刑种（行 367）——"充军属于五刑"= 错。

## 一句话口诀

> **"墨劓剕宫辟是旧伤；笞杖徒流死开皇定；文景拆桥，北齐铺路"**
