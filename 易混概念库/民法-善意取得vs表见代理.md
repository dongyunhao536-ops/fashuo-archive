# 善意取得 vs 表见代理

> 📌 **法硕"善意第三人保护"两大支柱的混淆点**。
> 题干都有"善意第三人"字眼，但保护机理完全不同。
> 答疑时凡是涉及"善意 + 第三人"且涉及让与/合同的题 → 第一步分清是物权变动场景还是代理场景。

---

## 一句话区分 test ★最重要★

> **善意取得 = 解决"无处分权人处分他人之物" 的物权归属问题**（无权处分人 → 善意受让人）
> **表见代理 = 解决"无代理权人以本人名义订立合同" 的合同效果归属问题**（无权代理人 → 善意相对人）

**判断流程**：先问争议焦点是"**物归谁**"还是"**合同对谁生效**"？
- 物的归属 → 善意取得（涉及《民法典》第 311 条）
- 合同效力归属 → 表见代理（涉及《民法典》第 172 条）

---

## 共通点

1. 都是为保护**善意第三人 / 善意相对人**而设的"特别保护制度"
2. 都涉及**外观信赖原则**（物权公示信赖 / 代理外观信赖）
3. 都以"善意 + 无过失"为核心主观要件
4. 都会让"无权人"的处分行为对**真正权利人**发生效力（牺牲真正权利人保护交易安全）

---

## 教材依据

### 善意取得
**《民法学》教材 第 2215-2232 行**（物权编 / 善意取得）

关键原文（第 2215-2221 行）：
> "善意取得，又称即时取得，是指无处分权人将他人动产或者不动产让与第三人或者为第三人设定他物权时，如果第三人在取得该动产或者不动产的物权时系出于善意且符合其他条件，即取得该动产或者不动产的所有权或者他物权的制度。"
>
> 动产善意取得四要件（第 2218-2221 行）：
> 1. 该项动产须为**占有委托物**且为非禁止流通物
> 2. 让与人系**无处分权人**
> 3. 受让人取得动产时出于**善意**（不知、无重大过失）
> 4. 受让人以**合理的价格**受让
> 5. 已经完成**交付**（不包括占有改定）
>
> 不动产善意取得四要件（第 2222-2224 行）：让与人有"权利外观" + 受让人善意 + 合理价格 + 已办**登记**

### 表见代理
**《民法学》教材 第 1752-1787 行**（代理 / 三、表见代理）

关键原文（第 1754-1758 行）：
> "表见代理是指行为人虽没有代理权，但第三人在客观上有理由相信其有代理权而与其实施法律行为，该法律行为的后果由本人承担的代理。表见代理在代理制度中起协调本人利益和相对人利益的作用，通过对善意第三人利益的保护而达到维护交易秩序的目的。"
>
> 构成四要件（第 1760-1771 行）：
> 1. 代理人**无代理权**（含超越授权、终止后继续代理）
> 2. 有**被授予代理权的外观**（介绍信、公章、空白合同书等）
> 3. 相对人**不知且无过失**（善意）
> 4. 相对人**基于信任而与无权代理人成立法律行为**

---

## 法条/司法解释

- **善意取得**：《民法典》第 **311 条**（动产/不动产善意取得四要件）；第 **312 条**（遗失物特别规则）
- **表见代理**：《民法典》第 **172 条**（行为人没有代理权、超越代理权或代理权终止后……相对人有理由相信行为人有代理权的，该代理行为有效）

## 一句话口诀

> **"善意取得管物权，表见代理管合同；公示信赖归前者，外观信赖归后者"**
>
> 或更直白版：
> **"物归善意人，约束本人受"**（善意取得让善意人得物；表见代理让本人受约束）

## 关键并列对照表

| 维度 | 善意取得 | 表见代理 |
|---|---|---|
| 制度功能 | 物权归属 | 合同效力归属 |
| 法条 | 《民法典》311 条 | 《民法典》172 条 |
| 处分人身份 | 无处分权人 | 无代理权人 |
| 受让/相对人身份 | 善意受让人 | 善意相对人 |
| 关键外观 | 占有 / 登记 | 介绍信 / 公章 / 授权书 |
| 主观要件 | 善意 + 无重大过失 | 善意 + 无过失（**法硕用语略严**） |
| 客观要件 | **合理价格** + 交付/登记 | **基于信任成立法律行为** |
| 法律后果 | 受让人取得物权，原权利人丧失物权 | 合同对本人有效，由本人承担后果 |
| 真正权利人救济 | 向无处分权人请求损害赔偿 | 向无权代理人请求赔偿 |

## 常见错误选项设计（题目陷阱模式）

1. **以"善意取得"作干扰**：题干是代理类（伪造印章、超越授权）→ 应选表见代理，**不**选善意取得
2. **以"表见代理"作干扰**：题干是物的转让（A 把 B 的物卖给 C）→ 应选善意取得，**不**选表见代理
3. **"合理价格"陷阱**：以"白菜价""明显低于市场"接手 → 排除善意取得（心得第4条）
4. **"无外观"陷阱**：无权代理人无任何授权迹象 → 不构成表见代理，构成狭义无权代理
5. **遗失物 / 漂流物 / 埋藏物**：原则上**不**适用善意取得（《民法典》312 条），但通过拍卖或经营资格者购得有特别规则

## 法考 vs 法硕立场差异

| 点 | 法考立场 | 法硕立场 | 备注 |
|---|---|---|---|
| 善意取得是否要"合理价格" | 是 | **严格要求**，是法硕反复挖坑点 | 心得第4条明确 |
| 占有改定能否善意取得 | 有学理讨论 | **不考** | 法硕只考要件齐备性 |
| 表见代理"善意 + 无过失" | "无重大过失"标准较多见 | **明确"无过失"**（教材原文） | 法硕用语更严 |

⚠️ **特别提醒**：法硕严格依《考试分析》/《民法学》原文表述。学理上的"善意第三人保护一元说"等观点，**不要带入答题**。
