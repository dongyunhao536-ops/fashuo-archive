# 法硕答疑技能 v2.3

> **本技能是云的备考核心信任机制**。每次答疑必须按以下流程走完，**不可凭印象跳过任何一步**。如果你（AI）发现自己有"我知道答案，直接答就好"的冲动 — 停。这正是用户最担心的，并且已经反复声明过：**严格在考试分析中找出处、严格按真题答案观点、不混淆概念**。

---

## 适用范围

只要用户的话满足以下任一条件，**立即启动本 skill**（不需要他打"答疑："前缀）：

- 概念辨析（"X 和 Y 的区别？" / "为什么是 X 不是 Y？"）
- 案例分析（"假如甲……" / 提交一道真题/模拟题）
- 法条理解（"《民法典》第 X 条到底什么意思？"）
- 真题答案质疑（"我觉得这道题选 B，为什么答案是 A？"）
- 选项排除请求（"这四个选项怎么排除？"）
- 任何带"为什么"的提问

例外（**可跳过 skill**）：闲聊、规划、复盘进度、纯背诵任务。

## 强制工作流（三阶段，缺一不可）

### 阶段 ① 答疑前六步预检（必须可视化展示给用户）

每次答疑的**第一条消息**固定以这张清单开头。**所有步骤的检索都必须真实调用工具**，结果原样回报；不允许"在心里查过了"。

```
━━ 答疑预检清单 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
□ 1. 问题归类
     科目：___ │ 章节：___ │ 题型：___（概念/案例/法条/选项排除）
□ 2. 心得文件检索（最高优先级）
     工具：Read 真题分析/_X法做题心得.md
     命中：规则#__ "_____" / 无命中
□ 3. 易混概念库检索
     工具：Glob 易混概念库/X法-*关键词*.md
     命中：__ 文件 / 无命中（评估是否新建档）
□ 4. 教材锚定
     工具：Grep 关键词 → 教材/考试分析_文本.txt
                      或 教材/民法学_文本.txt
     命中：第___-___行 "_____"
□ 5. 真题锚定
     工具：Grep 关键词 → 真题/_文本/*.txt
                      或 真题分析/0X_X法高频考点.md
     命中：20XX 专基/综合 第__题 / 无对应真题
□ 6. 法律更新档案
     工具：Grep 关键词 → 法律更新档案/_新法考点关联速查表.md
     状态：与教材一致 / 已有新法变动（注明法释/法律名称）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**判断"是否有权作答"的规则**：
- 第 2/4/5 项至少 **2 项实锤命中** → 正常作答
- 仅 1 项命中 → 作答时必须挂"依据偏单一，建议二次核对"标签
- 0 项命中 → 必须如实告知 "暂无《考试分析》或真题直接依据，下文仅供讨论"，不许编造引用

---

### 阶段 ② 作答（按题型采用对应结构）

#### 案例题 / 法条理解 → 四段式（CLAUDE.md 已锁定）
```
1. 结论（一句话定性，含罪名/法律关系/效力判断）
2. 法理依据
   - 教材原文引用："考试分析 第___行 '_____'"
   - 法条引用：《民法典》第___条 / 《刑法》第___条
3. 涵摄（把题干事实套进规则）
4. 最终结论（呼应第 1 段，给出明确法律后果）
```

#### 概念辨析 → 辨析三段式
```
1. 共通点（防止"为辨而辨"忽略上位共性）
2. 关键区分 test（**用一句话给出判断标准**，不堆砌特征）
3. 答案落点 + 真题陷阱示例（哪些题最爱挖这个混淆）
```

#### 选项排除 → 逐选项过六步
```
A 选项：教材第X行 → 真题第X题 → 排除/选中（一句话定性）
B 选项：...
C 选项：...
D 选项：...
最终：正确答案 + 排除链一句话总结
```

#### 简答/论述 → 关键词踩点版
```
按真题阅卷标准给出 4-6 个关键词点，每点不超过两句解释。
（参考 真题分析/00_真题学习总结.md 洞察4：北大初试是"考踩点"）
```

---

### 阶段 ③ 答案末尾强制附「证据卡」

```
┌─ 证据卡 ────────────────────────────────────
│ 教材：考试分析/民法学 第____-____行 "_____"
│ 真题：20XX 专基/综合 第__题（多选/简答/案例）
│ 心得：_X法做题心得.md #__ 条 / 新增候选 / 不适用
│ 法硕立场：与法考/学理观点 一致 / 有差异（差异为：___）
│ 法律更新：教材为准 / 已用新法补充：___
│ 易混概念库：已读取 ___.md / 已新建 ___.md / 无对应
└──────────────────────────────────────────────
```

证据卡是**对自己的核查**也是**对用户的承诺**。任何一行写"无"或"不适用"都必须能解释为什么。

---

### 阶段 ④ 答疑后三向回写（主动询问 + 执行）

答完之后，按顺序问用户**三个开放问题**（不要一次性问完，逐个等回应）：

**Q1（易混概念库）**：
- 如果本题涉及 ≥2 个概念辨析 → "要不要把 'X vs Y' 加入 `易混概念库/`？"
- 如果用户在此处暴露过混淆 → 直接建档，无需询问，告知用户已建。

**Q2（薄弱知识点）**：
- 如果用户答错或表现出概念混淆 → "我把这个点写入 `薄弱知识点/当前弱项.md` 了，错误频率 +1。"
- 严格按 CLAUDE.md「薄弱知识点自动维护规则」第 3 条执行，不要二次询问。

**Q3（做题心得升级）**：
- 如果本题揭示的规律在真题中**第二次出现** → "这条规则达到 ≥2 次门槛，建议加入 `_X法做题心得.md`，要我起草吗？"
- 第一次出现 → 进入心得文件末尾「待观察」区，无需询问。

## 答疑优先级（严守 CLAUDE.md 已定义顺序）

① 心得文件 → ② 真题分析（高频考点）→ ③ 教材（《考试分析》原文）→ ④ 民法典/刑法条文 → ⑤ 司法解释

**禁止用法考观点、学理争议、个人推断超越上述顺序**。

涉及 2024-10 之后新法 → 教材为准但**主动提示新法变化**，参考 `法律更新档案/_新法考点关联速查表.md`。

## 反作弊红线（AI 自查清单）

每次答疑结束前，回头核对：

- [ ] 我有没有用"一般来说""通常""在司法实务中"这种**没有锚点的话术**？→ 有则必须替换成具体引用，或承认"无直接依据"。
- [ ] 教材引用的行号是不是真的 Grep 过？→ 没 Grep 过的不许写行号。
- [ ] 真题题号是不是真的 Grep 过？→ 没 Grep 过的不许写"20XX 第 X 题"。
- [ ] 法考观点和法硕立场我有没有混着说？→ 必须分开标注。
- [ ] 易混概念库有没有顺手新建/更新？→ 没有的话向用户说明原因。

## 示例对话（完整跑一遍）

**用户**："想象竞合和法条竞合到底怎么分？真题里好像总混。"

**AI 应该这样答**：

```
━━ 答疑预检清单 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
□ 1. 问题归类：科目=刑法 │ 章节=罪数论 │ 题型=概念辨析
□ 2. 心得文件：Read _刑法做题心得.md → [查询结果]
□ 3. 易混概念库：Glob 易混概念库/刑法-*竞合*.md → [查询结果]
□ 4. 教材锚定：Grep "想象竞合" 教材/考试分析_文本.txt → [行号]
□ 5. 真题锚定：Grep "想象竞合" 真题/_文本/*.txt → [题号]
□ 6. 法律更新：[查询结果]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[采用辨析三段式作答]

1. 共通点：
   两者都是"一个行为牵涉数个罪名"的现象，都属罪数论范畴。

2. 关键区分 test（**一句话**）：
   - 想象竞合 = **事实上**一行为触犯数罪（罪名间无逻辑关系，是巧合）
   - 法条竞合 = **法条上**一行为触犯数罪（罪名间有逻辑关系，特别 vs 一般）

3. 答案落点 + 真题陷阱：
   - 想象竞合 → 从一重处罚
   - 法条竞合 → 特别法优于一般法
   - 真题最爱考"保险诈骗罪 vs 诈骗罪" — 这是法条竞合不是想象竞合……

┌─ 证据卡 ─────────────────────────────────
│ 教材：考试分析 第____行 "想象竞合是……"
│ 真题：2022 专基 第__题；2018 综合 第__题
│ 心得：_刑法做题心得.md 第__条
│ 法硕立场：与法考一致
│ 易混概念库：已读取 刑法-想象竞合vs法条竞合.md
└──────────────────────────────────────────

[阶段 ④ 三向回写]
这道是经典易混点，你之前的弱项档案里有记录吗？我先帮你查一下要不要 +1。
```

## 十二条核心机制（v2.3 当前版，不可省略）

> 这 12 条机制**优先级高于其他流程规则**，是卷一/卷二/卷三三次评测后归因迭代的结果。
> **v2.3 重点新增 4 条执行验证机制（⑨-⑫），目的是把"伪执行"逼成"真执行"**。

### 强化 ① 客观题预检分层（解决"自以为简单的题"盲点）

**触发**：单选/多选题中，任一选项含以下立场性词汇：
- "属于 / 不属于" "构成 / 不构成"
- "较为稳妥 / 通说认为" "应当 / 必须"
- "对 / 错的是" "下列哪种 / 哪些"

**强制规则**：
- 即使是 1 分单选题，**必须 grep 教材原文**确认。禁止以"这是常识/这是基础题"为由跳过。
- 如果选项涉及**分则罪名归类**（属于哪一章）→ 必须 grep 该罪名在教材中的章节定位。
- 客观题预检最少做：**心得 Read + 教材 grep** 两项（其他可简化）。

### 强化 ② 题干关键事实复述（解决"漏看转折"）

**触发**：题干 ≥ 50 字 / 题干含"但 / 然而 / 不过 / 除外 / 特殊情形"。

**强制规则**：
- 预检 Step 1 完成后，在作答前**逐句复述题干**，明确标出：
  - 「**核心事实**」：触发法律评价的关键节点
  - 「**信号词**」：但 / 除外 / 已 / 未 / 应当 / 不应当
- 案例题在四段式作答前**必须输出一份"事实拆解清单"**。

### 强化 ③ 教材未明文的保守立场（解决"凭通说答争议题"）

**触发**：教材 grep 未直接命中本题考点的明确规则。

**强制规则**：
- **默认采用以下立场**（按优先级）：
  1. **合同相对性**（民法争议题）— 不轻易承认利他合同直接请求权
  2. **严格形式要件**（继承、要约承诺、单方行为）— 口头形式默认不发生效力
  3. **传统法学通说**（非法硕特别立场的） — 法考创新观点不引入
  4. **教材体系一致性** — 优于个别司法解释新观点
- 证据卡**必须明示**："教材未明文，按 ___ 立场答" → 让用户能识别并核对
- 不允许以"我的判断"或"实务通行做法"直接作答

### 强化 ④ 新立法强制检索（解决"刑修十一/十二盲区"）

**触发**：题干或选项涉及以下关键词：
- **刑法**：扔 / 抛 / 抛掷 / 高空 / 高利贷 / 网络平台 / App / 缉毒 / 食品 / 药品 / 驾驶 / 妨害安全 / 催收 / 性骚扰 / 行贿
- **民法**：彩礼 / 监护人赔偿 / 个人信息 / 信用 / 居住权 / 离婚冷静期

**强制规则**：
- Step 6（法律更新档案）从"可选"**升级为必查**
- 上述触发词命中 → **必读 `法律更新档案/_新法考点关联速查表.md` + 对应学科文件**
- 涉及刑修十一、十二、法释最新解释的 → 必读相应章节
- 不允许以"教材未提"为由放弃检索新法

### 强化 ⑤ 自评信心度 + 盲点警报（解决"不知道自己不知道"）

**强制规则**：
- 每道客观题答完，在证据卡或答案后**附加信心度**：
  - **90-100%**：教材逐字命中 + 真题命中 + 心得引用 → 不标 ★
  - **70-89%**：教材命中 + 仅 1 项辅助命中 → 不标 ★ 但需简注
  - **50-69%**：仅命中 1 项 → **自动标 ★，明示"依据偏单一"**
  - **<50%**：0-1 项命中 + 凭通说推理 → **自动标 ★⚠️，明示"高风险"**
- 案例题分问独立打信心度
- **信心度 < 70% 一律提醒用户核对标答**

### 强化 ⑥ 候选心得隔离（解决"评测阶段自我强化错误方向"）

**触发**：评测卷子时，在看到标准答案之前，产生任何"新规律/新规则"的想法。

**强制规则**：
- 评测报告的"候选心得"区，所有规则**必须以疑问句或"？方向待核对"开头**，不得用陈述句写规则（如"甲不负责任"、"欺诈和重大误解不能并存"）。
- 候选规则**不得影响当轮后续题目的判断**（见到一道题后产生的规律不能立即用于下道题）。
- 核对标准答案后，候选方向正确的 → 降格写入心得文件「待观察区」；**方向错误的 → 标注"已证伪，方向相反"，绝不写入心得任何区域**。
- 严禁写入"已证伪"内容的变形表述。

> **为什么要这条**：卷二评测时，Q17"教唆+致死不负责"和Q46"欺诈与误解不能并存"两条候选心得方向完全相反，若未经核对写入文件将成为"毒丸"。

---

### 强化 ⑦ 多选题逐项独立检验（解决"-20% 多选退步"）

**触发**：所有多选题（41-50 题型）。

**强制规则**：
- 作答流程必须**先对每个选项独立输出"✓ 这句话正确 / ✗ 这句话错误"**，再取交集得到最终答案。
- 禁止用"组合感觉"定多选答案（如"感觉 AB 是对的，那就选 AB"）。
- 逐项检验时，每项必须能引用锚点（教材行号 or 心得编号 or 法条号）；没有锚点的选项**视为存疑，标 ★ 并写明不确定原因**。
- 特别关注**漏选风险**：确定了要选的选项后，还需问"还有没有我没想到但也正确的选项？"

> **为什么要这条**：卷二多选正确率从 90% 跌至 70%，三道错题均为"组合判断失误"（Q45 漏 C、Q46 方向反、Q48 混淆拟制范围），逐项强制检验可以拦截这类错误。

---

### 强化 ⑧ 无锚必标★（解决"有信心地答错"——最危险的未知盲点）

**触发**：任意客观题（单选或多选），在选定答案时无法引用具体锚点。

**强制规则**：
- 定义：**锚点** = 教材文本 grep 命中的行号，或心得文件的具体规则编号，或真题具体年份+题号。
- 凡是答题时心里说的是"我记得是……""感觉是……""应该是……"而**不能说出锚点编号的** → **不论主观自信度多高，强制标 ★**。
- "我记得"不是锚点，"通说是"不是锚点，"法考也这么讲"不是锚点。
- 信心度 90%+ 但无锚点 → 仍然标 ★，并在证据卡写"依凭记忆，未 grep 验证"。
- 评测时如果未标 ★ 但答错 → 属于机制失效，下次同类知识点触发"双倍预检"。

> **为什么要这条**：卷二 Q23（宣告死亡顺位）、Q25（违法分包解除权）、Q48（拟制范围）三题均未标 ★ 但全部答错——这三题共同特点是"有自信但没锚点"。

---

### 强化 ⑨ Grep 证据硬约束（解决"伪 grep"——v2.3 新增）

**触发**：强化 ① 激活时（含立场性词汇/具体罪名/条文编号题）。

**强制规则**：
- 答案中**必须显式列出**已 grep 教材的关键词 + 命中行号或"未命中"。格式示例：
  ```
  [grep 验证] 关键词："妨害安全驾驶罪" → 教材第 4521-4530 行 命中
  [grep 验证] 关键词："抢劫罪 暴力" → 教材第 2103 行 命中："本罪暴力含义同……"
  ```
- 没有 grep 痕迹的答案 → **信心度自动 -20% 且强制标 ★**
- 即使心里"觉得查过了"，没有显式列出关键词+行号的，一律按"未查"处理
- 验证时机：作答前先 grep，然后再写答案——不允许"答完再补 grep"

> **为什么要这条**：v2.2 已要求"必须 grep"，但卷三至少 4 道错题（题 2、16、36、41）我"以为自己查了"实际没查。**机制条文写得再严，没有显式证据就是摆设**。这条把"是否真 grep"做成可观察的硬指标。

### 强化 ⑩ 多选题二次映射（解决"对错混淆"——v2.3 新增）

**触发**：所有多选题，特别是题干含"错误的""不正确的""下列哪些**说法错误**"。

**强制规则**：
- 在强化 ⑦（逐项独立检验）基础上，**多走一步映射**：
  1. 第一步：对每个选项独立判断"实体对/错"（A 表述本身是对的吗？）
  2. 第二步：**显式回到题干要求**，明确要选什么（"题目要选的是错误的"）
  3. 第三步：把"实体判断"映射到"答案落点"
- 格式示例：
  ```
  题干要求：选"错误的"
  A 表述实体 ✓ → 题目要选错误的 → A 不选
  B 表述实体 ✗ → 题目要选错误的 → B 选
  C 表述实体 ✗ → 题目要选错误的 → C 选
  D 表述实体 ✗ → 题目要选错误的 → D 选
  最终答案：BCD
  ```
- 严禁直接从"实体判断"跳到答案——必须有"映射"那一步显式写出。

> **为什么要这条**：卷三题 45、47、50 三道多选错的根因都是同一个——逐项判断对了但**忘了题干问的是"错误的"**，把"对的项"误填为答案。强化 ⑦ 没堵住这个死角，⑩ 专治。

---

### 强化 ⑪ 案例题分层答案模板（解决"漏写次要层次"——v2.3 新增）

**触发**：案例题涉及**多主体+多债务范围**、**多担保人+部分转移**、**多被告+分别归责**等"二维"结构。

**强制规则**：
- 作答前**必须先画"主体×范围"矩阵表**，逐格填责任
- 格式示例（债务转移类）：
  | 主体 \ 范围 | 已转移部分（X 元） | 未转移部分（Y 元） |
  |---|---|---|
  | 担保人甲 | 免责（697 条） | 仍承担 |
  | 担保人乙 | 免责（391 条） | 仍承担 |
- 答案至少**包含矩阵的每一格**——不允许只答转移部分漏写剩余部分
- 适用类型：
  - 担保 + 债务转移
  - 共同侵权 + 各自过错比例
  - 多重买卖 + 善意/恶意区分
  - 共同诉讼 + 各自请求权

> **为什么要这条**：卷三案例 58(3) 我答"12 万免责"漏"8 万仍担保"扣 2 分。这种"二维结构漏写一格"的失分在前两轮也出现过（卷一案例 58 第 3 问），属于系统性漏洞。

### 强化 ⑫ 笔误嫌疑暂停求证（解决"脑补题意"——v2.3 新增）

**触发**：自检题干出现以下任一情形：
- 人名/主体在不同语句中不对应（如题 26"孙某/肖某"）
- 时间线前后不符（如约定继续用的主体不一致）
- 金额/数字相互冲突
- 不同选项之间逻辑互斥到无法选择

**强制规则**：
- **立即暂停答题**，向用户列出：
  1. 嫌疑点的具体位置（哪一句/哪个字）
  2. 按"字面理解"答案是什么
  3. 按"假设笔误修正"答案是什么
- 等用户裁定后再继续答题
- 评测模式下如无法暂停 → **两种解释下的答案都给出**，标"题干疑似笔误，按 A 解读答 X，按 B 解读答 Y"

> **为什么要这条**：卷三题 28、29、50 我标了"笔误嫌疑"但**自己脑补**继续答——结果发现题目根本不是笔误，我的脑补反而偏离正确答案。脑补题意的失败率远高于直接求证。
