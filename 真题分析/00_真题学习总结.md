## 二、12年真题的核心洞察

### 洞察1：法硕（非法学）是"全国统考"，北大不能改命题
你的目标是北大，但**专业课题目就是这24份真题的延续**——出题逻辑、题型结构、考点分布每年都很稳定。所以：
- **24份真题 = 你最重要的"教材"之一**
- 任何不在真题考过的"考点"，价值都低于已经反复出现的考点
- 北大主要看复试，初试拿到235+/300（专业课）就能进复试

### 洞察2：题型已经稳定（2018年后基本不变）
近8年（2018-2025）的题型几乎没动过：
- 专业基础：40单选 + 10多选 + 4简答 + 2法条 + 2案例
- 综合：40单选 + 10多选 + 3简答 + 3分析 + 2论述
- 这意味着我们可以**精确预测2026年的卷面结构**

### 洞察3：高频考点高度集中
基于12年统计，约**30%的考点**在真题中重复出现≥3次。这些考点就是性价比最高的复习目标：
- **刑法**：共犯、罪数、正当防卫、因果关系、停止形态
- **民法**：合同效力、担保物权、侵权责任、人格权、继承
- **法理**：法的本质、法律关系、法律解释、法的价值、法系
- **宪法**：国家机构、公民基本权利、立法体制、备案审查
- **法制史**：唐律疏议、明清会审、清末修律、革命根据地

### 洞察4：北大初试不是"考记忆"，是"考踩点"
分析题、案例题、论述题的评分都是**关键词踩点**。比如简答"没收财产与罚金的区别"，答出"内容/适用方式/执行方式/适用对象"四个角度，每个4-6字关键词即可拿全分。
- 不是写得多就有分
- 是**关键词覆盖+逻辑清晰**就有分

### 洞察5：你最薄弱的领域恰恰是最高频的
对照 CLAUDE.md 中你提到的薄弱项：
| 你的薄弱点 | 真题中频次 | 警示 |
|---|---|---|
| 刑法分则罪名区分 | 每年案例必考 | **必须攻克** |
| 民法物权编 | 每年6-8题 | **必须攻克** |
| 民法合同编 | 每年8-10题 | **必须攻克** |
| 宪法国家机构 | 每年5-7题 | **必须攻克** |
| 立法法 | 每年3-5题 | **必须攻克** |

**好消息**：薄弱点恰好对应高频考点，攻破它们就是最高ROI的复习。

## 三、对你的复习建议（基于真题）

### 阶段1：基础打通（建议2-3个月）
1. **刑法+民法**：通读《考试分析》，搭建知识框架
2. **法理+宪法+法制史**：教材通读后立即结合真题
3. **不需要做模拟题**，先把12年真题精做一遍

### 阶段2：真题精做（建议2个月）
策略：**按科目纵向刷**，不要按年份横向刷
1. 把12年刑法题集中刷（按年份顺序）→ 错题进错题本
2. 民法、法理、宪法、法制史同理
3. 重点对比"近三年（23-25）" vs "早期（14-18）"的命题风格变化

### 阶段3：主观题强化（建议1-2个月）
1. 背诵12年简答题答案 → 形成自己的"简答模板库"
2. 法条分析：训练"逐句拆解+延伸论证"的答题习惯
3. 案例分析：训练"行为定性→罪数判断→刑罚情节"的答题流程
4. 论述题：背诵5-6个"万能段落"（法治价值、依宪治国、民本思想等）

### 阶段4：冲刺整合（考前1个月）
1. 错题本完整回顾
2. 当年的政治时事+法治热点（近1年最高法工作报告、立法热点）
3. 模拟限时考试（每周一套）
