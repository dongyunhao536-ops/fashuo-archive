# 民法学高频考点（2014-2025真题归纳）

> 在专业基础卷中占75-90分（选择20+多选≈5+简答20+法条10+案例15）

## 一、民法总则

### ⭐⭐⭐ 必考核心

#### 1. 民事法律行为效力 **（每年6-8道选择题集中考查）**
- **有效**：意思表示真实+主体适格+不违法不违背公序良俗
- **无效**：通谋虚伪（黑白合同 — 2025-31）、违反强制规定、违反公序良俗
- **可撤销**：重大误解、欺诈、胁迫、显失公平、乘人之危 **（2024-37胁迫、2024-25重大误解、2022-21显失公平vs可撤销）**
- **效力待定**：限制行为能力人、无权代理、无权处分 **（2025-39误装合同材料、2024-29冒用公章、2024-25未成年充值）**
- **不成立**：合意不达 **（2024-27代金券）**

#### 2. 民事主体
- **自然人**：行为能力、监护、宣告失踪/死亡 **（2024-21宣告死亡、2025法条56监护指定）**
- **法人**：营利/非营利/特别法人；法人独立责任 **（2025-35子公司独立法人）**
- **非法人组织**：合伙、个独、不具法人资格的服务机构 **（2025-33）**
- **农村承包经营户**（2024简答53）

#### 3. 代理 **（高频）**
- **委托代理 vs 法定代理 vs 指定代理**
- **表见代理**（2017简答54、2024-29无授权伪造印章）
- **滥用代理权**（自己代理、双方代理、损害被代理人利益）**（2015简答54、2023-28隐瞒报价）**
- **无权代理与追认**（2024-29追认前撤销权）
- **代理权范围限制不得对抗善意相对人**（2022法条56）

#### 4. 意思表示
- 明示 / 默示 / 沉默
- **意思表示的解释**：误载不害真意（2025-39）
- 拟制承诺 vs 沉默承诺 vs 明示承诺（2022-22 寄送商品）

#### 5. 诉讼时效
- 一般时效3年
- 特别时效
- 时效中断、中止
- 期间起算

## 二、物权

### ⭐⭐⭐ 物权变动 — 学生薄弱核心区
- **不动产**：登记生效（房屋买卖、抵押），登记对抗（特殊动产）
- **动产**：交付生效，特殊动产对抗登记主义
- **基于法律行为 vs 非基于法律行为**（继承、判决、征收、公益收回 — 2025多选50B）
- **预告登记**（2018法条56）
- **居住权登记取得**（2025-36朋友设居住权）

### 担保物权 ⭐⭐⭐
- **抵押**：不动产登记生效、动产抵押登记对抗（2025-34未登记抵押权未设立、2024多选46）
- **质押**：动产质押交付生效（2024-40转质押、2025-34）
- **留置**（2024多选49修理留置）
- **浮动抵押**（2025简答54确定情形、2021法条56、2023多选46）
- **保证**（一般保证 vs 连带保证；未约定从一般 — 2025-34D）
- **抵押期间转让规则**（2023-25未登记不得转让约定 vs 已登记）

### 用益物权
- **土地承包经营权**（设立时间-合同生效时 — 2022-23）
- **宅基地使用权**（2021简答53）
- **建设用地使用权**（提前收回的补偿 — 2025多选50）
- **居住权**（2024多选47特征、2025-36）
- **地役权**（2019简答53）

### 所有权
- **共有**（按份/共同；共有物分割 — 2023法条56）
- **业主的建筑物区分所有权**（加装电梯 — 2025-28、2023-22电梯广告费）
- **善意取得**（2017案例60）
- **添附**（2025-21换轮胎）
- **拾得遗失物**（2024多选48）
- **占有**（直接/间接占有）

## 三、合同

### ⭐⭐⭐ 高频核心

#### 1. 合同成立与生效
- 要约与承诺（实质性变更条款 — 2024-35乙报价更改）
- 合同成立时点（2025-26离婚协议未办理登记的财产分配）
- 格式条款（未提示注意 — 2022-30 不成为合同内容）
- 黑白合同（2025-31）

#### 2. 合同履行
- **同时履行抗辩权 / 先履行抗辩权 / 不安抗辩权**（2023-37拒绝交货抗辩权、2024-24先履行抗辩、2014简答53不安抗辩）
- 第三人代为履行（2025-24具有合法利益的第三人）
- 涉他合同（向第三人履行、由第三人履行）

#### 3. 违约责任
- 违约金酌减（2025-23 — 不可单方约定不调整）
- 违约金 vs 定金（择一）
- 缔约过失 vs 违约责任（2024-35、2023-13）
- **预约合同**（违反预约的责任 — 2025多选46）
- 违约责任与侵权责任竞合（2024-39 选择请求权）

#### 4. 合同变更与解除
- 合同变更（双方协商，单方无效 — 2025-21A厂换轮胎）
- 解除权行使
- 显失公平≠合同显著低价（2023案例58 标价错误）

#### 5. 债的关系
- **代位权诉讼**（2025-37 B公司诉C公司）
- **债权转让**（2024-22 转让通知后债务人不能向原债权人付款）
- **债务承担**（2025-27 转租≠债务承担）
- **抵销/混同/免除/提存**（2022-29 连带债务）
- **以物抵债协议**（2025-29 诺成合同，自合意时成立）
- **不当得利**（2023-36 善意得利返还）
- **无因管理**（2022-32 情谊行为不构成）

#### 6. 各类有名合同
- **买卖**：分期付款、试用买卖、所有权保留（2022多选46 售后回租）
- **租赁**：转租（2025-27 经同意转租）；优先承租权（2025-27D 同等条件优先）
- **赠与**：撤销权（2019简答54、2016案例60）
- **借款合同**：实践合同vs诺成合同
- **建设工程合同**：承包人优先受偿权（2019法条56）
- **保管、运输、行纪、居间合同**
- **技术合同**：职务发明（2015简答53）

- **夫妻共同财产 vs 个人财产**（2025-32 婚前财产）
- **夫妻共同债务**（2024-30 个人借款不写入共债）

### 继承
- **法定继承顺序**（2025多选47）

- **遗产管理人**：遗嘱执行人 → 继承人推选 → 协商不成共同担任 → 民政部门或村委（2023-27）

## 八、薄弱点警示（基于 CLAUDE.md 你的弱项：物权、合同）

### 物权易混点（每年都考）
1. 抵押期间转让的处理（约定不得转让，未登记不能对抗第三人 — 2023-25）
2. 不动产登记效力 vs 对抗效力 — 这是基础但易错
3. 共有物分割与瑕疵分担（2023法条56）
4. 业主决议（多数决，少数业主补偿 — 2025-28）

### 合同易混点
1. 黑白合同：黑（真实）有效，白（虚假）无效（2025-31）
2. 预约合同 vs 本约合同（违反预约≠违反本约）（2025多选46）
3. 以物抵债协议 — 诺成合同，自合意成立（2025-29）
4. 第三人代为履行的"合法利益"边界（2025-24 租赁期满乙无利益）
5. 违约金酌减规则（高于损失过分时，但不能约定不可调整 — 2025-23）

### 侵权易混点
1. 共同饮酒：劝酒+未照顾=过错；未劝+尽到照顾=无过错（2025案例58）
2. 学校责任：直接侵权人主责+学校未尽职=补充责任（2024-32）
3. 自甘风险 vs 受害人故意 vs 意外事件（2023-38碰瓷）
