## 41. 答案：AB（交通肇事罪 + 伪造货币罪不能由单位构成）

**锚点**：单位犯罪须**法律明文规定**——
- ✓ A 交通肇事罪：自然人犯罪，**不能**由单位构成（法律未规定单位主体）；
- ✓ B 伪造货币罪：**不能**由单位构成（仅自然人）；
- ✗ C 妨害信用卡管理罪：**可**由单位构成（刑法 177 条之一第 3 款明文）；
- ✗ D 侵犯公民个人信息罪：**可**由单位构成（刑法 253 条之一第 4 款明文）。
**信心度**：75%

## 42. 答案：ABD（任意+事前+一般共犯，望风非简单共犯）

**逐项**：
- ✓ A 任意共犯：法律未将共犯作为构成要件的犯罪（盗窃罪可一人也可多人）→ 任意共犯；
- ✓ B 事前共犯：商量好后实施 → 事前通谋的共同犯罪；
- ✗ C 简单共犯：简单共犯=共同正犯，要求**全部参与实行行为**——王某仅望风（非实行行为）→ **不是简单共犯，是复杂共犯**；
- ✓ D 一般共犯：非有组织/集团性 → 一般共犯（与特殊共犯/犯罪集团相对）。
**坑警示**：望风通常是帮助行为而非实行行为，属于复杂共犯（有分工）。
**信心度**：80%

## 43. 答案：ABD（不作为故意杀人既遂+无因果关系+作为杀人未遂均错）★

**题目要求"错误的"**：
- 案情：甲（父）儿子乙掉河中 → 甲不救+开枪射击乙 → 乙怕被打死躲水中过久窒息死 → 甲误以为击毙；
- ✗ A "甲成立不作为故意杀人既遂"——错（错误项即答案）：甲有作为的杀人故意（开枪），不作为只是先前同时存在，但乙实际死因是窒息（自己躲水中），与甲不作为救助的因果关系可成立；但与"积极射击"想象竞合——按一罪处理，定性为**作为故意杀人**而非"不作为既遂"；
- ✗ B "甲射击行为与乙死亡不存在因果关系"——错：因果关系存在（甲射击直接导致乙躲水中过久 → 窒息），属介入因素未中断因果；
- ✓ C "甲发生了因果关系认识错误"——对：甲以为乙是中弹死亡，实际是窒息死亡，属**因果关系错误**（同一构成要件内的偏差，不影响既遂）；
- ✗ D "甲成立作为故意杀人罪未遂"——错：甲对死亡结果有故意+实际死亡结果发生+具因果关系 → **既遂**而非未遂。

⚠️ 取错误项：ABD。
**信心度**：60% ★（A 项不作为定性存在争议）

## 46. 答案：AB（继续履行+解除+赔偿）★

**题目要求"正确的"**：
- 锚点：民法典第 502 条第 2 款——合同**报批义务**条款（**独立有效**）：
  - 即使合同未批准生效，但合同中"申请批准"等条款仍**独立有效**；
  - 报批义务人不履行 → 相对人可请求**继续履行**或**解除合同**并赔偿损失。
- ✓ A 陈某可请求继续履行审批义务——502 条 2 款明文；
- ✓ B 陈某可解除合同并请求赔偿损失——502 条 2 款明文；
- ✗ C 该合同已经生效——错：合同**未经审批未生效**（502 条 1 款），仅报批条款独立有效；
- ✗ D 陈某不能请求解除合同——错：可以解除（B 已确认）。

**取正确项**：AB。
**信心度**：60% ★

## 47. 答案：BCD（高空抛物责任分配）

**题目要求"错误的"**：
- 锚点：民法典第 1254 条——高空抛物坠物责任：
  - ① 能确定具体侵权人 → **侵权人承担责任**；
  - ② 经调查难以确定具体侵权人 → 由**可能加害的建筑物使用人给予补偿**，但**能够证明自己不是侵权人的除外**；
  - ③ 物业服务企业等建筑物管理人未尽安全保障义务 → 承担未履行职责的责任。
- 题干：无法查明抛物者+物业知道经常有人扔垃圾置之不理+赵某证明当时家中无人；
- 逐项判断：
  - ✓ A "由可能加害的建筑物使用人补偿+物业承担与其过错相应的补充责任"——正确（命中 1254 条）→ **A 不是错误项**；
  - ✗ B "由物业公司承担与其过错相应的责任，其余部分由可能加害的建筑物使用人给予适当补偿"——错：应是可能加害人补偿为主+物业为补充责任，B 顺序颠倒 → 错误项；
  - ✗ C "由可能加害的建筑物使用人赔偿"——错：法条是"**补偿**"非"赔偿" → 错误项；
  - ✗ D "赵某须对刘某承担责任"——错：赵某已**证明自己不在家**（不是侵权人）→ 不承担 → 错误项。

**取错误项**：BCD。
**信心度**：80%

## 48. 答案：AB（环境修复+惩罚性赔偿）

**锚点**：民法典第 1234 条+1232 条——
- 1234 条：违反国家规定造成生态环境损害，**国家规定机关或法律规定组织有权请求修复**；
- 1232 条：违反国家规定**故意污染环境破坏生态**造成严重后果的，可请求**惩罚性赔偿**。
- 题干：化工公司将废液交无资质方处理+山上倾倒+严重污染 → 故意/明知违法 → 命中惩罚性赔偿；
- 逐项：
  - ✓ A 须承担环境修复责任（1234 条）；
  - ✓ B 可请求惩罚性赔偿（1232 条）；
  - ✗ C 化工公司**有过错**（明知吴某良无资质仍交付）→ 须担责；
  - ✗ D 环境侵权举证：**由污染者证明其行为与损害之间不存在因果关系**（1230 条举证责任倒置）→ 不是由原告证明。

**取正确项**：AB。
**信心度**：80%

## 49. 答案：AC（3.8 取得+3.18 抵销 80 万）

**锚点**：
- 民法典第 545-547 条债权转让：
  - **债权转让自合同成立时（让与人与受让人之间）生效**——3 月 8 日顺风公司**已取得**对如意公司的债权（A 对）；
  - **未经通知，对债务人不发生效力**——通知是对债务人的对抗要件——3 月 10 日通知到达如意公司，自此对如意公司生效；
- 民法典第 549 条抵销：
  - 债务人在收到债权转让通知时对让与人享有债权+先于债权到期或同时到期 → 可向受让人主张抵销；
  - 如意公司对吉祥公司有 80 万装修款债权（已到期）+ 在 3 月 10 日通知前/后均可主张抵销；
  - C 对：3 月 18 日如意公司可主张抵销 80 万。
- 逐项：
  - ✓ A 3 月 8 日顺风取得债权（让与人受让人间生效）；
  - ✗ B 3 月 10 日不是顺风取得债权的时点（取得是 3.8）；
  - ✓ C 3 月 18 日如意可抵销 80 万；
  - ✗ D 如意可以抵销（与 C 矛盾）。

**取正确项**：AC。
**信心度**：70%

## 50. 答案：ACD（价款超优先+无权处分+质押合同效力）★

**题目要求"错误的"**：
- 案情：何某购甲设备 100 万，约定一年后付款，**付款完毕前甲公司保留所有权**；3.1 甲交付；3.3 何某将设备出质给乙公司+交付借款 50 万；3.8 何某和甲办理"保留所有权登记"（视为价款超级优先权登记 416 条）；到期何某无力清偿，乙拟拍卖。
- 锚点：
  - 民法典第 641 条：保留所有权约定**未经登记，不得对抗善意第三人**；
  - 民法典第 416 条：标的物交付后 **10 日内登记**的价款抵押权，优先于其他抵押权——本题甲 3.8 登记，距 3.1 交付 7 日，10 日内 → 享有价款超级优先权；
  - 但乙公司是**质押**（动产质押，3.3 交付即成立质权，无需登记）；
  - 质押与抵押的优先顺序：通常**先成立者优先**——乙 3.3 已成立质权（先），甲 3.8 才登记（后）；
  - **教材立场**：416 条价款超优先权仅明文优先于浮动抵押和其他抵押权，对于**质权**是否优先**教材未明文** → 按保守立场，**先成立的质权优先**。

**逐项**：
- ✗ A "对设备价款甲公司优先于乙公司受偿"——错（教材未明确价款抵押优先于先设质权，且乙质权先成立）→ 错误项；
- ✓ B "对设备价款乙公司的质权优先于甲公司受偿"——正确（先成立者优先）→ 非错误项；
- ✗ C "何某设立质权是有权处分"——错：保留所有权未登记**不得对抗善意第三人**，乙善意取得质权——但**何某出质时无所有权**（甲保留所有权），属**无权处分**+乙善意取得 → C 表述"有权处分"错误 → 错误项；
- ✗ D "质押合同无效"——错：质押合同**有效**（合同效力与处分权无关，合同有效但需善意取得才能产生质权）→ 错误项。

**取错误项**：ACD。
**信心度**：50% ★（题干"出质给乙公司"与"保留所有权登记"时序+教材立场结合较复杂）
