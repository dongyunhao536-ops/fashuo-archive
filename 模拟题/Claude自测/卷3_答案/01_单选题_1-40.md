## 1. 答案：B（扩大解释）

**坑警示**：A 文理（按字面）；C 缩小（外延变小）；D 类推（超出可能含义，罪刑法定禁止）。
**信心度**：85%

## 2. 答案：B（贪污罪/受贿罪"利用职务便利"相同解释）

**锚点**：教材立场——贪污罪和受贿罪都属第八章贪污贿赂罪，二者"利用职务上的便利"含义**完全相同**：利用本人职务上主管、管理、经手公共财物或具体执行公务的便利。
**排除**：
- A 错——正当防卫"必要限度"以制止侵害为准；紧急避险"必要限度"要求损害小于所避免损害；
- C 错——抢劫罪暴力可至致死；劫持航空器罪暴力为对人身强制，**含义不同**；
- D 错——非法侵入住宅的"侵入"是物理进入；非法侵入计算机信息系统的"侵入"是技术性突破访问控制，**含义不同**。
**信心度**：70%

## 3. 答案：C（故意或过失）

**锚点**：食品、药品监管渎职罪（刑法第 408 条之一）——教材归为渎职罪章节，渎职罪通常兼含故意（滥用职权）与过失（玩忽职守）两种主观形态。该罪条文表述"滥用职权或者玩忽职守" → 故意或过失均可。
**坑警示**：D"故意+徇私动机"是徇私舞弊型滥用职权要件，本罪不限于此。
**信心度**：60% ★（食药监管渎职罪冷门）

## 29. 答案：A（丙公司的抵押权优先于乙农商银行的抵押权）★

- 方某经济富裕 → 不存在"无力抚养" → 不属法定送养人（A 对）；
- B 错：方某哥哥（华侨）有 3 名子女——收养人**应当无子女或只有 1 名**（1098 条）→ 哥哥不能收养；
- 但 A 已锁死答案——方某无资格送养，整个送养行为不成立；
- C 错：何某父母对孙抚养义务不等于"送养优先权"；
- D 错：送养应征求被送养人父母双方意见。

⚠️ A 和 B 均能成立法律点，但 A 更直接锁送养资格，作为单选答案 A 更稳。
**信心度**：50% ★

## 34. 答案：D（蔡某承担责任后，可以在马某的过错范围内向其追偿）

**锚点**：民法典第 1189 条——**委托监护**：
- 蔡某是法定监护人；委托马某照看 → 委托监护；
- 小蔡致害他人 → **由蔡某承担侵权责任**；
- 受托人马某有过错的 → 承担相应责任（追偿权）；
- 教材立场：委托监护责任主体仍为法定监护人，受托人过错时承担"相应责任"。
- A、B、C 均不符："按份责任"（A 错）、"全部+补充"（B 错）、"马某承担全部"（C 错）。
**信心度**：65% ★

## 35. 答案：B（乙公司承担责任）

**锚点**：民法典第 1191 条——**用人单位责任**：员工因执行工作任务造成损害，由用人单位承担。
- 丙是乙清洁公司员工+操作不当致害+执行工作 → **乙公司承担**；
- 甲物业仅作为**定作人**（民法典 1193 条承揽关系：定作人对承揽人执行任务造成损害一般不承担，除定作/指示/选任有过错）；
- 甲物业仅委托具有专业资质的乙清洁公司 → 无过错 → 不承担；
- D 错：丙是乙员工，由乙承担，丙不独立担责。
**信心度**：85%

## 36. 答案：A（方某没有解除权）

**锚点**：民法典第 587 条解约定金——
- 给付定金方放弃定金可不履行（解约定金）；
- 但本题方某**尚未支付定金** → 解约定金权未实际产生；
- 甲公司无违约 → 方某无任意解除权 → A 对；
- B 错：甲公司本无违约，亦无定金可主张罚则（未交付）；
- C 错：方某无解除权；
- D 错：解约定金需放弃，不能要求返还自己的定金。
**信心度**：80%

## 37. 答案：C（于某承担保证责任后可以向赵某追偿）★

**锚点**：
- 题干"无条件承担保证责任"——明示放弃先诉抗辩权 → 推定为**连带责任保证**（A、B 对）；
- C 错：教材立场——保证人之间**没有约定时无追偿权**（共同保证内部互不追偿，仅可向债务人追偿，民法典 700 条主流解读）；
- D 错：连带责任保证未约定保证期间按 **6 个月**（692 条）→ 表述对。

⚠️ 教材立场对共同保证人之间是否互有追偿权存在争议：民法典 519 条连带债务人间内部追偿存在，但担保解释明确**共同保证人之间未约定追偿的不互相追偿**。**C 错（错误项即答案）**。
**信心度**：75%

## 38. 答案：C（东兴幼儿园承担补充责任）★

**锚点**：民法典第 1201 条——**第三人侵权+教育机构未尽管理职责**：
- 5 周岁小张属**无民事行为能力人**；
- 第三人（社会混混）侵权+幼儿园未尽管理职责（保安疏忽+监控故障）；
- 民法典 1201 条："**由第三人承担侵权责任**；**教育机构未尽到管理职责的，承担相应的补充责任**"；
- C 对（补充责任）；
- D 错——"先行承担"不符合补充责任的法律定性。

⚠️ 修正卷评测报告答案 D → 应为 **C**（补充责任）。
**信心度**：55% ★
