## 41. 答案：BCD

**题干**：罪责刑相适应原则的说法哪些正确？

**逐项**：
- A **错**——"刑法不溯及既往"是**罪刑法定原则**的派生要求（事先明文规定），不是罪责刑相适应原则
- B 对——罪责刑相适应要求立法上**制定合理的刑罚体系**（轻重有序）
- C 对——刑罚与犯罪性质、犯罪情节、罪犯人身危险性相适应（教材原文表述）

**坑警示**：罪刑法定 vs 罪责刑相适应——前者要求"事先明文+不溯及既往"，后者要求"罚当其罪+体系合理"。

## 42. 答案：BC

**题干**：属于犯罪构成的**选择性要素**的有？

**逐项**：
- A **错**——犯罪故意和过失（主观罪过）是**必备要素**（任何犯罪都有罪过形式，要么故意要么过失）
- B 对——犯罪动机是**选择性要素**（仅在少数罪名中为构成要件，如某些贪利动机）
- C 对——危害结果是**选择性要素**（行为犯不要求结果，结果犯要求结果）
- D **错**——危害行为是**必备要素**（任何犯罪都必须有危害行为，无行为则无犯罪）

## 43. 答案：AB

**题干**：同类客体是**社会主义市场经济秩序**的犯罪有？

**锚点**：刑法分则同类客体——
- 第三章 **破坏社会主义市场经济秩序罪**：洗钱罪、非国家工作人员受贿罪、走私（一般）罪、合同诈骗罪等
- 第六章 **妨害社会管理秩序罪**：掩饰隐瞒犯罪所得罪（第一节扰乱公共秩序罪）、走私毒品罪（第七节毒品犯罪）

**逐项**：
- A 对——非国家工作人员受贿罪在第三章第三节（妨害对公司、企业管理秩序罪）
- B 对——洗钱罪在第三章第四节（破坏金融管理秩序罪）
- C **错**——掩饰、隐瞒犯罪所得罪在**第六章第二节妨害司法罪**（不属于市场经济秩序）
- D **错**——走私**毒品罪**在**第六章第七节毒品犯罪**（与一般走私罪分章，不属于市场经济秩序）

**坑警示**：D 易误选——"走私"字样并不代表归第三章。走私毒品/淫秽物品/武器弹药等分别归不同章节。
**信心度**：93%

## 44. 答案：ABC

**题干**：下列说法哪些选项是**错误**的？

**逐项**：
- A **错**——化工厂"**所有的人员**"均构成重大环境污染事故罪——单位犯罪原则上只处罚**直接负责的主管人员和其他直接责任人员**，不是所有员工。**A 错** ✓
- B **错**——刑法**有规定**单位可构成污染环境罪（第 346 条：单位犯本节罪的，对单位判处罚金，并对其直接负责的主管人员和其他直接责任人员，依照本节各该条的规定处罚）→ 单位**可以**构成污染环境罪 → B 表述错误 ✓
- C **错**——甲是环保局监察大队长（**特定主体**：负有环境保护监督管理职责的国家机关工作人员）→ 应定**环境监管失职罪**（第 408 条特别法），不是玩忽职守罪（第 397 条一般法），按特别法优于一般法。C 表述错误 ✓
- D 对——环境监管失职罪+受贿罪，按"渎职+受贿数罪并罚"规则（**心得 #2**：国家机关工作人员渎职+受贿必须数罪并罚）

**罪名认定**：本题考的就是"渎职罪+受贿罪并罚"——心得 #2 直接命中。
**法律更新档案**：受贿罪——刑修十二+法释〔2026〕6 号已检索，不影响本题（受贿罪入罪标准仍适用）。
**信心度**：94%

## 45. 答案：AB

**题干**：关于单位犯罪的说法，下列**正确**的有？

**逐项**：
- A 对——为实施犯罪而设立的单位（"皮包公司"型），按**自然人犯罪**处理，**不认定单位犯罪**（最高法司法解释明文）
- B 对——国有公司高管集体决定将单位资金分给个人 → 本质是**共同贪污罪**（贪污"私分国有资产"型），不是单位犯罪（私分国有资产罪的主体是单位但处罚仅及直接责任人）

  注：本题表述"在 5 个高管中平均分配"——如果是私分国有资产罪（第 396 条），主体是单位且处罚直接责任人；但 50 万元×5 = 总额超出单位犯罪门槛，**且分配给个人 = 共同贪污**——结论仍是不认定单位犯罪。

- C **错**——单位被兼并/合并/分立后实施单位犯罪——按司法解释，**仍以原单位为被告**追究原单位责任人，**承继单位（乙公司）**仅在承继财产范围内承担**罚金责任**，**不承担刑事责任**（不能追究乙公司刑事责任）
- D **错**——甲个人实施拐卖儿童罪+借用单位车辆和司机——单位**仅是被利用**的工具，**不构成单位犯罪**（不是单位决策实施）

**信心度**：90%

## 46. 答案：BD

**题干**：第三人欺诈情形的判断。

**案情解读**：甲被丙（第三人）欺骗，将画卖给丁。
- 关键：丁是否知道丙的欺诈？题干 B 选项明示"**丁不知道丙实施欺诈**"

**法理**：第三人欺诈（《民法典》第 149 条）——
- 第三人实施欺诈，**对方知道或应知**该欺诈 → 受欺诈方有权撤销
- 第三人实施欺诈，**对方不知** → 受欺诈方**不能撤销**

**逐项**：
- A **错**——丁不是受欺诈方，没有撤销权；甲是受欺诈方
- B 对——丁不知欺诈+合同成立时丁是相对人 → 甲不能以欺诈为由撤销
- C **错**——重大误解和欺诈在民法理论上**不能并存**（同一事实中），甲的错误认识由丙的告知引起，不是"自己单方误解" → 不构成重大误解
- D 对——丁不知欺诈+不构成重大误解 → 甲**不能撤销**买卖合同

**信心度**：80% ★（C 是难点——是否能转化为重大误解。法硕教材立场是不能并存，按 D 对处理。如果标答有 C，需更新心得"被第三人欺诈+相对人不知，可转化为重大误解"）

## 47. 答案：AD

**题干**：预约转本约的判断。

**锚点**：预约转本约规则（《最高法关于审理买卖合同纠纷案件适用法律问题的解释》第 2 条）——
- 当事人签订认购书等预约合同，**主要条款已经具备**（标的、价款、履行等）+ **一方已经履行主要义务，对方接受** → 可以认定**本约合同已成立**

**题干信号**：
- "别墅交付的情况、价款、登记过户时间等均有明确约定" ✓ 主要条款齐备
- "杨某支付了购房款"（**全部**）+ "甲公司接受" ✓ 一方履行主要义务+对方接受
→ 本约合同**已成立**

**逐项**：
- A 对——本约合同已成立
- B **错**——认购书在此情形下**已转化为本约**，不再仅是预约合同
- C **错**——甲不再仅承担违反预约的责任
- D 对——甲应承担违反**本约合同**的违约责任（继续履行/赔偿损失等）

**信心度**：85%

## 48. 答案：ACD

**题干**：下列表述**错误**的是？

**锚点**：《民法典》第 18 条第 2 款——16 周岁以上的未成年人，**以自己的劳动收入为主要生活来源**的，**视为完全民事行为能力人**。
- 小曾 17 岁 + 餐厅打工有固定收入 + **足以养活自己** → 视为**完全民事行为能力人**

**逐项**：
- A **错**——小曾视为**完全民事行为能力人**，不是限制民事行为能力人 ✓
- B 对——小曾视为完全行为能力人 → 自己承担侵权责任（《民法典》第 1188 条仅适用于无/限制民事行为能力人）
- C **错**——小曾视为完全行为能力人 → 自己承担全部责任，**不存在父亲补充支付**的问题 ✓
- D **错**——小曾完全民事行为能力人 + 已成年 → 自己被告，不应追加曾某 ✓

**坑警示**：题问"错误"，A/C/D 都错，仅 B 正确。

## 49. 答案：ABC

**题干**：购车后未交付单证的法律后果。

**锚点**：
- 汽车是**特殊动产**，所有权变动**以交付为准**（《民法典》第 224 条），登记仅对抗善意第三人（第 225 条）
- 单证（发票、合格证）是**从给付义务**
- 从给付义务的不履行若**导致合同目的不能实现** → 可以**解除合同**（《民法典》第 563 条）+ 司法解释支持

**逐项**：
- A 对——乙已交付车 → 甲已取得所有权（特殊动产以交付转移）
- B 对——单证是从给付义务，甲有权要求乙交付
- C 对——一直拒绝交付单证，致使甲无法办理登记和牌照（涉及**汽车更新指标 2018 年 2 月 28 日到期**——影响甲使用车辆的根本合同目的）→ 可以主张合同解除（《买卖合同司法解释》第 6 条）
- D **错**——本题情形下，由于影响合同目的（更新指标即将到期+无法上牌+无法使用），甲**可以**主张合同解除

**信心度**：80% ★（C/D 二选，按"从给付义务影响合同目的可解除"立场选 C 对、D 错）

## 50. 答案：AB

**题干**：摄影服务的合同性质与解除权。

**逐项**：
- A 对——影楼写真服务（拍照、化妆、相册制作等）是**承揽合同**（《民法典》第 770 条：承揽人按照定作人的要求完成工作，交付工作成果，定作人支付报酬。包括加工、定作、修理、复制、测试、检验等工作）
- B 对——《民法典》第 787 条：**定作人可以随时解除承揽合同**，造成承揽人损失的，应当赔偿损失。两人作为定作人**有权单方解除**
- C **错**——单方**变更**合同需对方同意（变更和解除是不同的形成权——解除是法定形成权，变更没有相应法定单方权）
- D **错**——两人有权单方解除（同 B）

**坑警示**：定作人有"任意解除权"是承揽合同的**特别权利**，但变更合同没有这种"任意变更权"。
**信心度**：90%
