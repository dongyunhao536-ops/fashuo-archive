# 卷一·多项选择题（41-50）答案

> 答题模式：批量紧凑版
> 多选规则：至少有 2 个选项正确，少选/错选/多选不得分

## 刑法多选（41-45）

### 41. ABCD
**理由**：刑修十二行贿罪从重处罚情形：
- A 向司法工作人员行贿 → 命中"对监察、行政执法、司法工作人员行贿"
- B 向多人行贿 → 命中"多次行贿/向多人行贿"
- C 用违法所得（贩毒资金）行贿缉毒警察 → 命中"将违法所得用于行贿" + "向司法工作人员行贿"
- D 国家工作人员行贿 + 谋取职务晋升 → 命中"国家工作人员行贿" + "为谋取职务晋升行贿"
**锚点**：法律更新档案/01_刑法相关更新.md（刑修十二行贿从重情形）。

### 42. AC（不属于危险犯的）
**理由**：
- **A 生产、销售、提供劣药罪 → 结果犯**（要求实际危害结果，与假药罪不同）
- B 投放危险物质罪 → **具体危险犯**（属于危险犯）
- **C 故意杀人罪 → 结果犯/实害犯**（要求死亡结果方为既遂）
- D 生产、销售、提供假药罪 → 刑修十一改为**抽象危险犯**（属于危险犯）
**锚点**：考试分析 第 2269-2293 行（危险犯定义及典型分类）+ 危害公共安全罪章节。

### 43. ABD（侮辱罪、暴力干涉婚姻自由罪、诽谤罪）
**理由**：我国刑法亲告罪 5 类：**侮辱罪、诽谤罪、暴力干涉婚姻自由罪、虐待罪、侵占罪**（口诀："侮诽辱婚虐侵"）。
- A 侮辱罪 ✓
- B 暴力干涉婚姻自由罪 ✓（致人死亡的除外）
- C 遗弃罪 ✗（不是亲告罪）
- D 诽谤罪 ✓（严重危害社会秩序和国家利益的除外）
**锚点**：考试分析 妨害婚姻家庭罪 + 妨害名誉罪章节。

### 44. ABCD
**理由**：四罪都要求"为谋取不正当利益"：
- A 行贿罪（第 389 条）
- B 对有影响力的人行贿罪（第 390 条之一）
- C 斡旋型受贿罪（第 388 条，"通过其他国家工作人员的职务行为，为请托人谋取不正当利益"）
- D 利用影响力受贿罪（第 388 条之一，"通过其他国家工作人员的职务行为，为请托人谋取不正当利益"）
**锚点**：考试分析 第 9421-9622 行（贪污贿赂罪章节中行贿/受贿对应罪名）。

### 45. AD
**理由**：
- **A 对**：通过微信群组织抢红包式赌博、设规则、持续组织 → **开设赌场罪**（"利用网络等方式建立赌博平台"）
- **B 错**：为境外赌博网站代理、开设公众号接受投注（"情节严重"）→ 直接构成**开设赌场罪**（正犯，非帮助犯）—— 法释〔2005〕3号
- **C 错**：组织境内的**外国人**到境外赌博 → 不构成"组织参与国（境）外赌博罪"——该罪对象为"**中国公民**"
- **D 对**：通过 App 组织数百中国公民境外赌博 → **组织参与国（境）外赌博罪**（刑修十一新增，刑法第 303 条第 3 款）
**锚点**：考试分析 妨害社会管理秩序罪/赌博罪章节。

## 民法多选（46-50）

### 46. AC
**理由**：
- **A 对**：张某是办公室主任（**职务代理人**），以甲公司名义签合同，构成职务代理 → 合同对甲公司生效（《民法典》第 170 条）
- B 错（合同**已成立并生效**）
- **C 对**：《民法典》第 170 条第二款："法人或者非法人组织对**执行职务行为的限制**，**不得对抗善意相对人**"——反向解读：相对人**非善意**（知道限制）则可对抗，甲公司可拒付款
- D 错（合同有效，非无效）
**锚点**：民法学教材 第 170 条（职务代理）。

### 47. CD
**理由**：
- **A 错**：3/5 乙公司同意但**要求派人前来签订合同书** → 双方约定以**书面形式签订为合同成立要件**（《民法典》第 490 条），合同尚未成立
- **B 错**：3/12 杨某拒绝签约，此时甲公司**尚未履行主要义务**，不能要求乙签字盖章
- **C 对**：3/15 甲转账注明"购买普洱茶货款"，乙公司**接受并使用该笔款项** → 适用《民法典》第 490 条第二款"**一方履行主要义务，对方接受时，合同成立**"
- **D 对**：合同成立后甲有权请求乙交付货物
**锚点**：民法学教材 合同编 第 490 条（合同成立特殊规则）。

### 48. ACD
**理由**：
- **A 对**：《民法典》第 1254 条第一款："**可能加害的建筑物使用人给予补偿**" — 是补偿（非赔偿）
- **B 错**：归责原则是**过错推定**（被告需证明自己不是加害人才免责），不是普通过错责任
- **C 对**：《民法典》第 1254 条第二款："**物业服务企业等建筑物管理人**应当采取必要的安全保障措施防止前款规定情形的发生" — 即有安全保障义务
- **D 对**：《民法典》第 1254 条第三款："发生本条第一款规定的情形的，**公安等机关应当依法及时调查，查清责任人**"
**锚点**：民法学教材 侵权责任编 第 1254 条（建筑物抛掷物致害）。

### 49. ACD
**理由**：
- **A 对**：信用错误影响个人社会评价 → 侵犯**名誉权**（《民法典》第 1024 条："名誉是对民事主体的品德、声望、才能、**信用**等的社会评价"——信用是名誉权的内容之一）
- **B 错**：我国**没有独立的"信用权"**作为具体人格权，信用属名誉权范畴
- **C 对**：可请求赔礼道歉（名誉权侵权救济）
- **D 对**：《民法典》第 1029 条："民事主体可以依法查询自己的信用评价；发现信用评价不当的，**有权提出异议并请求采取更正、删除等必要措施**"
**锚点**：民法学教材 人格权编 第 1024、1029 条。

### 50. AC
**理由**（教材民法学第 5996-5997 行**逐字命中**）：
> "教唆人、帮助人承担侵权人应承担的**全部责任**；监护人**在未尽到监护职责的范围内与教唆人、帮助人共同承担责任**"

- **A 对**：教唆人李四承担"侵权人应承担的全部责任" → 王五可要求李四承担全部
- **B 错**：教材第 5992-5994 行明确："教唆人、帮助人以其**不知道且不应当知道**行为人为限制行为能力人为由，主张**不承担侵权责任或者仅承担连带责任**的，**人民法院不予支持**" → B 的"连带"说法不成立
- **C 对**：张三作为监护人，"在未尽到监护职责的范围内与李四共同承担责任" — 教材逐字命中
- **D 错**：张小三 17 岁兼职（题干"经常以自己的劳动收入为主要生活来源"未提到，所以应认定为**限制行为能力人**），侵权时身份决定责任主体，**起诉时已成年不改变监护人责任**

**锚点**：民法学教材 第 5985-6000 行（教唆/帮助限制行为能力人侵权的规则，最高法司法解释吸收）。
