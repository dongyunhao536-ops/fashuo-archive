## 模拟题录入规范

每道模拟题独立 .md 文件，命名：
```
YYYY-MM-DD_{核心标识}.md
```
例如：`2026-06-03_甲乙控制丙索财案.md`

文件内必含字段：
1. **题目原文**（一字不改）
2. **出题来源**（自出 / 用户 / 某模拟卷）
3. **科目 + 章节 + 题型**
4. **参考答案**（按四段式 / 辨析三段式 / 关键词踩点版）
5. **关键考点**（含教材锚点行号）
6. **易混点**（关联易混概念库文件）
7. **答题表现记录**（云的实际答题情况、用时、错误环节）
8. **关联心得 / 真题母题**
9. **复盘日期**（首次答 / 二次回顾 / 三次稳定）

## 维护规则

1. **触发录入**：用户提交任意模拟题（自己出/培训机构/网络）→ 即时归档
2. **触发更新**：用户二次回顾、错题重做 → 更新"答题表现记录"字段
3. **触发提炼**：每累计 10 道 → 生成高频考点归纳
4. **触发清理**：错误反复 ≥3 次的考点 → 同步登记到 `薄弱知识点/当前弱项.md`
5. **答疑联动**：「法硕答疑」skill 第 5 步真题锚定时，**同时搜索本目录**（模拟题也算"已见过的题"）
